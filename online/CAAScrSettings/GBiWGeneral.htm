GBiWGeneral Setting Repository
Quick Reference

GBiWGeneral

Role: These setting parameters define BIW fastening application parameters

Detailed role: Enable the user to specify the BIW fasetners parameters

The GBiWGeneral repository has the following characteristics:

  • Primary Tab Page: /Shape/AutomotiveBiWFastening/General
  • Repository File: GBiWFastening
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

GBiWGeneral setting repository attributes:

  • AllowJointBodyCreation
    Role: : Manage joint body creation after OK on joint command
    • Type: boolean
    • Default value: true
  • AngularTolerance
    Role: : Specify angular tolerance for drafting extraction
    • Type: double
    • Default value: 90
  • BfmDefaultSetUpPath
    Role: Specifies the directory that contains the fastener standard to use
    • Type: URN
    • Default value:
  • BfmExportDir
    Role: Selects the directory that will contain the export file
    Detailed role: Enables the user to select the output directory that will contain the export file
    • Type: URN
    • Default value: UNIX: ${CATTemp} Windows: ${CATTemp}
  • BfmReportDir
    Role: Selects the directory that will contain the generated report
    Detailed role: Enables the user to select the output directory that will contain the generated report
    • Type: URN
    • Default value: UNIX: ${CATTemp} Windows: ${CATTemp}
  • CheckConsistencyStd
    Role: : Check standard file consistency at import time
    • Type: boolean
    • Default value: false
  • EnoviaData
    Role: : Indicates if Enovia VPM unsaved data creation is allowed
    • Type: boolean
    • Default value: false
  • FixedSpotSize
    Role: Specify the fixed spot size
    • Type: double
    • Default value: 0
  • FixedSpotSizeCheck
    Role: : Indicates if fixed spot size is used during drafting extraction
    • Type: boolean
    • Default value: false
  • ImportSetUp
    Role: : Indicates if standard file is imported automatically
    • Type: boolean
    • Default value: true
  • LockIDCheck
    Role: : Indicates if fastener identifier is locked
    • Type: boolean
    • Default value: false
  • ReverseDirection
    Role: : Indicates if drafing view reverse direction is included
    • Type: boolean
    • Default value: false
