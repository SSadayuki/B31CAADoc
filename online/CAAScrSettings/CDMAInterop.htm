CDMAInterop Setting Repository
Quick Reference

CDMAInterop

Role: Interop settings

Detailed role: Setting to handle Interop parameters

The CDMAInterop repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/ProductStructure/ENOVIAVpm
  • Repository File: CDMAInterop
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: CDMAInteroperability
    • Interface_CPP: CATICDMASettingCtrl
    • LateType: CATCDMASettingCtrl

CDMAInterop setting repository attributes:

  • AllowReplaceStructExposedDuringExtractSave
    Role: Allow replace of structure exposed during extract/save.
    • Type: boolean
    • Default value: false
  • AskBeforeOverwriteExistingParts
    Role: Ask before overwriting existing parts.
    • Type: boolean
    • Default value: false
  • AutomaticHighlightInPSN
    Role: Automatic highlight in PSN
    • Type: boolean
    • Default value: true
  • CATCDMPercentCacheSize
    Role: Maximum number of files.
    • Type: integer
    • Default value: 80
  • CheckExistInVPMBeforeFBDI
    Role: Check existence in VPM before FBDI.
    • Type: boolean
    • Default value: false
  • DisableAppObjMgt
    Role: Disable applicative object management
    • Type: boolean
    • Default value: false
  • DisableOverwriteReloadWarnMsg
    Role: Disable the overwrite/reload warning message.
    • Type: boolean
    • Default value: false
  • DisableVPMSaveCommitPanel
    Role: Disable the VPM save commit panel.
    • Type: boolean
    • Default value: false
  • DisableVPMSaveReportPanel
    Role: Disable the VPM save report panel.
    • Type: boolean
    • Default value: false
  • EnableDirectV5VaultSave
    Role: Enable Direct V5 Vault save.
    • Type: boolean
    • Default value: false
  • ForbidInconsistentSave
    Role: Forbid inconsistent save.
    • Type: boolean
    • Default value: false
  • GFBDI
    Role: Generic FBDI Activation
    • Type: boolean
    • Default value: true
  • LoadAllVPMProperties
    Role: Load all EnoviaVpm properties
    • Type: boolean
    • Default value: false
  • ManualCacheCleaning
    Role: Manual cache cleaning.
    • Type: boolean
    • Default value: false
  • MappingFile
    Role: Mapping for generic FBDI.
    • Type: string
    • Default value:
  • NbFilesForDeletion
    Role: Number of files for deletion.
    • Type: integer
    • Default value: 200
  • NbFilesMax
    Role: Maximum number of files.
    • Type: integer
    • Default value: 500
  • NeverOverwriteExistingParts
    Role: Never overwrite existing parts.
    • Type: boolean
    • Default value: false
  • NoDBConnection
    Role: Disable CATIAV5 VPM DB Connection.
    • Type: boolean
    • Default value: false
  • VPMCachePath
    Role: VPM cache path.
    • Type: string
    • Default value:
  • WorkWithVPMCache
    Role: Work with the VPM cache system
    • Type: boolean
    • Default value: false
