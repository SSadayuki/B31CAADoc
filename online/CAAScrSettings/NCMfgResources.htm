NCMfgResources Setting Repository
Quick Reference

NCMfgResources

Role: Lists the settings of the resources used in the Machining applications

Detailed role: The resources machining settings are useful to manages options on tools and ressources catalogs

The NCMfgResources repository has the following characteristics:

  • Primary Tab Page: /Manufacturing/Resources
  • Repository File: NCMfgResources
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

NCMfgResources setting repository attributes:

  • FeedAutoUpd
    Role: Automatic compute of feedrate attributes of the operation
    Detailed role: Automatic compute of feedrates attributes of the operation
    • Type: boolean
    • Default value: true
  • MajorWordListLocation
    Role: MajorWordListLocation
    Detailed role: MajorWordListLocation
    • Type: string
    • Default value:
  • NCMfgNoToolInMachProcess
    Role: Interactive Tool selection if no tool is found
    Detailed role: Interactive Tool selection if no tool is found
    • Type: boolean
    • Default value: false
  • NCMfgToolQueryInMachProcess
    Role: Tool Query mode in Machining Process instanciation
    Detailed role: Tool Query mode in Machining Process instanciation
    • Type: integer
    • Default value: 0
    • Default Value Role: By default an Automatic Tool Query is done for Machining Process instanciation
    • Values:
      0 Automatic Tool Query for Machining Process instanciation
      1 Interactive Tool selection in case of multiple results forMachining Process instanciation
  • PPToolsLocation
    Role: Defines the directory where to find catalogs and files for tools and PP tables
    Detailed role: Defines the directory where to find catalogs and files for tools and PP tables
    • Type: URN
    • Default value: UNIX: ${CATStartupPath} Windows: ${CATStartupPath}
  • SpindleAutoUpd
    Role: Automatic compute of spindle attributes of the operation
    Detailed role: Automatic compute of spindle attributes of the operation
    • Type: boolean
    • Default value: true
  • SpindleDisplayArrow
    Role: Spindle Arrow
    Detailed role: Display spindle arrow
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: Violet Blue in RGB order
    • Min value: 0
    • Max value: 255
  • SpindleDisplayArrowHead
    Role: Spindle Rotation Arrow Head
    Detailed role: Display spindle rotation arrow head
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: Violet Blue in RGB order
    • Min value: 0
    • Max value: 255
  • Tool2DVisu
    Role: Tool preview after selection
    Detailed role: Tool preview after selection
    • Type: boolean
    • Default value: true
  • ToolAutoQuery
    Role: Automatic query after modification
    Detailed role: Automatic query after modification
    • Type: boolean
    • Default value: true
  • ToolGageBehavior
    Role: Total lenght linked to tool gage
    Detailed role: Total lenght linked to tool gage
    • Type: boolean
    • Default value: false
