DMU2DSettings

Role: DMU 2D Settings

The DMU2DSettings repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/DMU 2D Workshop
  • Repository File: DMU2DSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DMU2DSettings setting repository attributes:

  • BackgrdColor
    Role: Background color

    • Max value: 255
  • BackgrdMode
    Role: Color background
    • Type: integer
    • Default value: 0
