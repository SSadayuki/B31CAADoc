PERTConfig Setting Repository
Quick Reference

PERTConfig

Role: Tab page to handle the Compare Tool command Setting parameters

The PERTConfig repository has the following characteristics:

  • Primary Tab Page: /DPManufacturing/Pert Chart
  • Repository File: PERTConfig
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
