CATDesSettingDisplay Setting Repository
Quick Reference

CATDesSettingDisplay

Role: Settings to manage Imagine And Shape Display Options

The CATDesSettingDisplay repository has the following characteristics:

  • Primary Tab Page: /Shape/Imagine And Shape/Display
  • Repository File: CATDesSetting
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATDesSettingDisplay setting repository attributes:

  • BMColor
    Role: color of the Imagine and Shape Base Mesh
    • Type: unsigned integer
    • Default value: 255
    • Min value: 0
    • Max value: 255
  • BMLineThickness
    Role: Thickness of the Imagine and Shape Base Mesh
    • Type: integer
    • Default value: 1
  • BMLineType
    Role: Type of the Imagine and Shape Base Mesh
    • Type: integer
    • Default value: 1
  • BMLineWeightThickness
    Role: Thickness of the Imagine and Shape weighted Line of the Base Mesh
    • Type: integer
    • Default value: 3
  • BMLineWeightType
    Role: Type of the Imagine and Shape weighted Line of the Base Mesh
    • Type: integer
    • Default value: 1
  • BMPointSymbol
    Role: Type of the Imagine and Shape Base Mesh vertices
    • Type: integer
    • Default value: 5
  • BMPointWeightSymbol
    Role: Type of the Imagine and Shape Base Mesh weighted vertices
    • Type: integer
    • Default value: 4
  • BMSpecificSelColor
    Role: specific color for Selected Elements of the Base Mesh
    • Type: unsigned integer
    • Default value: 255
    • Min value: 0
    • Max value: 255
  • CurveColor
    Role: color of the Imagine and Shape Curve
    • Type: unsigned integer
    • Default value: 255
    • Min value: 0
    • Max value: 255
  • CurveThickness
    Role: Thickness of the Imagine and Shape Curve
    • Type: integer
    • Default value: 2
  • CurveType
    Role: Type of the Imagine and Shape Curve
    • Type: integer
    • Default value: 1
  • HTCoordMode
    Role: allow to define coordinates show mode (none, on mouse, on handle)
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 2
  • HTTextLevel
    Role: allow to define a level (none / short : long) for the Help Text
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 2
  • IsBMEnlargedDotSymbol
    Role: allow to define largest dot symbols
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • IsBMFaceFillOn
    Role: allow to define if highlighted face is filled
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • IsBMLineHiddenOn
    Role: Hidden lines are displayed as dashed
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • IsBMMultiColorOn
    Role: allow to define a specific color for each Base Mesh
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • IsBMSpecificSelColorOn
    Role: allow to define a specific color for Selected Elements
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • IsHiddenLink
    Role: allow to hide link
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • IsHTPinned
    Role: allow to define if Help Text is pinned on screen or on mouse pointer
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • SurfaceColor
    Role: color of the Imagine and Shape Subdivision Surface
    • Type: unsigned integer
    • Default value: 255
    • Min value: 0
    • Max value: 255
