DNBWKIWorkRenderSettings Setting Repository
Quick Reference

DNBWKIWorkRenderSettings

Role: Tools Options of DPM Work - Work Instruction Text 3D Display

Detailed role: Font settings and attributes are specified here for 3D Text Annotation

The DNBWKIWorkRenderSettings repository has the following characteristics:

  • Primary Tab Page: /DPManufacturing/DNBWork/Work Instruction Text 3D Display
  • Repository File: DNBWKIWorkRenderSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: DNBWorkInterfaces
    • Framework_IDL: DNBWorkInterfaces
    • Interface_CPP: DNBIWKIWorkRenderSettingAtt
    • Interface_IDL: None
    • LateType: DNBWKIWorkRenderSettingCtrl

DNBWKIWorkRenderSettings setting repository attributes:

  • DNBAttributeList
    Role: Specifies the Font options - StrikeThrough,..
    Detailed role: Value: 1 for checked, 0 for no-check. Array Index Details - [1]Strikethrough [2]Superscript [3] Overline [4] Subscript [5] 3D-Display:Parallel To Screen [6] 3D-Display:Zoomable
    • Type: integer
    • Default value: 1
  • DNBComboAttrList
    Role: Specifies the Font details
    Detailed role: Array Index Details - [1]Style-Regular,Bold..[2]Underline [3,4,5] Color R,G,B
    • Type: integer
    • Default value: 255
  • DNBFontName
    Role: Name of the Font
    Detailed role: Name of the Font - Default is "Monospac821 BT". Also tested for value "Algerian (TrueType)".
    • Type: URN
    • Default value: Monospac821 BT
  • DNBFontSize
    Role: Specifies the Font Size
    Detailed role: Specifies the Font Size - default value is [0.0035]. Also tested for value 0.010.
    • Type: double
    • Default value: 0.0035
  • DNBWINumParam
    Role: Specifies the dimension of DNBComboAttrList
    Detailed role: Specifies the dimension of DNBComboAttrList
    • Type: unsigned integer
    • Default value: 5
