VPMImpactGraph Setting Repository
Quick Reference

VPMImpactGraph

Role: Setting to handle the VPM Graph Properties

Detailed role: Setting to handle the VPM Graph Properties

The VPMImpactGraph repository has the following characteristics:

  • Primary Tab Page: /ENOVIA V5 VPM/VPM Impact Graph Properties
  • Repository File: VPMImpactGraph
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

VPMImpactGraph setting repository attributes:

  • CATImmCstButton
    Role: Option to see constraints in the imact graph
    • Type: boolean
    • Default value: true
  • CATImmCtxButton
    Role: Option to see contextual link in the imact graph
    • Type: boolean
    • Default value: true
  • CATImmMultiLineButton
    Role: Option to show node names on multiple lines
    • Type: boolean
    • Default value: false
  • CATImmPubButton
    Role: Option to see publications in the impact graph
    • Type: boolean
    • Default value: false
  • CATImmRefRefButton
    Role: Option to see document to document links in the imact graph
    • Type: boolean
    • Default value: true
  • CATImmSeperatorEditor
    Role: Separator character for the multiline mode
    • Type: string
    • Default value: |
  • CATImmTPButton
    Role: Option to see technological packages in the imact graph
    • Type: boolean
    • Default value: true
