MeasureSettings Setting Repository
Quick Reference

MeasureSettings

Role: Setting to handle the Measure parameters

The MeasureSettings repository has the following characteristics:

  • Primary Tab Page: /General/Parameters and Measure/Measure tools
  • Repository File: MeasureSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

MeasureSettings setting repository attributes:

  • CheckBoxDisplay
    Role: Setting to display the fill inside the box
    • Type: integer
    • Default value: 0
    • Values:
      0 The box will appear filled
      1 The box will be without fill
  • CheckTilde
    Role: Setting to display the tilde sign for approx measures
    • Type: integer
    • Default value: 1
    • Values:
      0 The tilde sign will not appear
      1 The tilde sign will appear
  • CheckUpdateInertiaPart
    Role: Activate automatic measure inertia update in part
    • Type: integer
    • Default value: 0
    • Values:
      0 Measure Inertia Update Deactivated
      1 Measure Inertia Update Activated
  • CheckUpdateInertiaProd
    Role: Activate automatic measure inertia update in product
    • Type: integer
    • Default value: 0
    • Values:
      0 Measure Inertia Update Deactivated
      1 Measure Inertia Update Activated
  • CheckUpdatePart
    Role: Activate automatic measure update in part
    • Type: integer
    • Default value: 0
    • Values:
      0 Measure Update Deactivated
      1 Measure Update Activated
  • CheckUpdateProd
    Role: Activate automatic measure update in product
    • Type: integer
    • Default value: 0
    • Values:
      0 Measure Update Deactivated
      1 Measure Update Activated
  • ColorParam
    Role: Default color of the text box
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Green in RGB order
    • Min value: 0
    • Max value: 255
  • MeasureOnGeometryType
    Role: Setting for measure on geometry type
    • Type: integer
    • Default value: 0
    • Values:
      0 Canonical type radio will be selected
      1 Geometry type radio will be selected
  • MeasureOnlyShownElementsStatus
    Role: Setting to display measure for shown elements
    • Type: integer
    • Default value: 0
    • Values:
      0 Measuring all elements
      1 Measuring shown elements
  • TextColorParam
    Role: Default color of the text of the measure
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: White in RGB order
    • Min value: 0
    • Max value: 255
  • WidthParam
    Role: Default width of line which define the measure
    • Type: integer
    • Default value: 1
    • Default Value Role: 0.13 mm
    • Min value: 1
    • Max value: 55
