Quick Reference

RenderingSettingsSatellites

Role: Photo Studio options

Detailed role: Full explanation of the Setting Role

The RenderingSettingsSatellites repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/Photo Studio/Satellites
  • Repository File: RenderingOptionsSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

RenderingSettingsSatellites setting repository attributes:

  • SatelliteComment
    Role: Comment on satellite computers
    • Type: string
    • Default value:
  • SatelliteCount
    Role: Count of satellites
    • Type: integer
    • Default value: 0
  • SatelliteCPU
    Role: CPU count of satellite computers
    • Type: string
    • Default value: 1
  • SatelliteMode
    Role: Enable distributed rendering
    • Type: boolean
    • Default value: false
  • SatelliteName
    Role: Name of satellite computers
    • Type: string
    • Default value:
  • SatelliteOS
    Role: OS of satellite
    • Type: string
    • Default value:
  • SatelliteUseLocal
    Role: Use local host cpu
    • Type: boolean
    • Default value: true
