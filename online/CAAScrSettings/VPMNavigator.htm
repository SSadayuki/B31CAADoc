VPMNavigator Setting Repository
Quick Reference

VPMNavigator

Role: Setting to handle the VPM Navigator properties

Detailed role: Setting to handle the VPM Navigator properties

The VPMNavigator repository has the following characteristics:

  • Primary Tab Page: /ENOVIA V5 VPM/VPM Navigator
  • Repository File: VPMNavigator
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

VPMNavigator setting repository attributes:

  • AutoShowNoShow
    Role: Check this option if you want to automatically hide instances saved uint NoShow state
    • Type: boolean
    • Default value: false
  • BackgroundColor
    Role: Selects the background color of VPM Navagator windows.
    Detailed role: Selects the background color of VPM Navagator windows in detailed role
    • Type: unsigned integer
    • Default value: 207
    • Default Value Role: Gray in RGB order
    • Min value: 0
    • Max value: 255
  • CaptureSelect
    Role: Automatically select nodes satisfying filter criteria
    • Type: boolean
    • Default value: false
  • Delete_SO
    Role: Authorize object deletion with impacted objects
    • Type: boolean
    • Default value: false
  • DocHoverInfo
    Role: Check this option if you want to show the attached Document Information when it is Expand in VPM Navigator
    • Type: boolean
    • Default value: false
  • DocVersioning
    Role: Checks the maturity rules before creating a new revision of a document
    • Type: boolean
    • Default value: false
  • EnablePubSub
    Role: Launches the Publish/Subscribe functionality. Activates a daemon that periodically checks the new publications
    • Type: boolean
    • Default value: false
  • ExpandOnOpen
    Role: Uncheck this option if you do not want the PRC to be automatically expanded when it is opened in VPM Navigator
    • Type: boolean
    • Default value: true
  • LockUnlockAvailability
    Role: Lock/Unlock availability
    • Type: integer
    • Default value: 2
    • Values:
      1 Only Easy Lock/Unlock functionality is available on VPM Nav tree
      2 Only standard Lock/Unlock functionality is available on VPM Nav tree
      3 Easy as well as standard Lock/Unlock functionality is available on VPM Nav tree
  • LogonAtStart
    Role: Launches the logon to the LCA database at the start
    • Type: boolean
    • Default value: false
  • MergeMode
    Role: Merges Part instances/ document revisions
    • Type: boolean
    • Default value: false
  • NodeColors
    Role: NodeColors role
    • Type: boolean
    • Default value: false
  • saveactionmod
    Role: Save : Display Action/Modification Chooser
    • Type: boolean
    • Default value: false
  • SendEmailToNewOwner_Choice
    Role: Send email to new owner (Transfer Ownership)
    • Type: boolean
    • Default value: false
  • ShowSpecsOnFilter
    Role: Unckeck this option if you do not want to see the Specification table while defining a Configuration Filter
    • Type: boolean
    • Default value: true
  • StableOrdering
    Role: The VPM Navigator graph nodes are ordered by type and alias
    • Type: boolean
    • Default value: false
  • sync
    Role: Synchronization rules for instances
    • Type: integer
    • Default value: 0
    • Values:
      1 Propagate synchronization to all displayed instances
      0 Always prompt for propagation to all displayed instances
      2 Never propagate to other instances
  • syncactionmod
    Role: Display Action/Modification Chooser
    • Type: boolean
    • Default value: false
  • TimerInterval
    Role: Enters in minutes the frequency of the Publish/Subscribe daemon
    • Type: integer
    • Default value: 180
  • treelayout
    Role: Tree Orientation
    • Type: integer
    • Default value: 0
    • Values:
      0 None
      1 Displays the VPM Nav tree in Vertical orientation
      2 Displays the VPM Nav tree in horizontal orientation
  • Versioning
    Role: Checks the maturity rules before versioning
    • Type: boolean
    • Default value: false
  • VPMNavRefreshNoProdInstExpand
    Role: Do not apply expand on Product Instance node on VPM Nav
    • Type: boolean
    • Default value: false
  • VPMNavVolumeFilterWithoutProdInst
    Role: Exclude Product Instances From volume Filtering on VPM Nav
    • Type: boolean
    • Default value: false
