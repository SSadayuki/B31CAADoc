CATPrtPropertyEditor Setting Repository
Quick Reference

CATPrtPropertyEditor

Role: Setting to manage the options of PartDesign

Detailed role: PartDesign setting contains options necessary to the commands of PartDesign

The CATPrtPropertyEditor repository has the following characteristics:

  • Primary Tab Page: /Mechanical Design/PartDesign
  • Repository File: Part
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATPrtPropertyEditor setting repository attributes:

  • PRT_Settings_NonManifold_Warning
    Role: Setting to manage the display of warning message for Non-manifold input
    Detailed role: Warning is displayed for Non-manifold input
    • Type: integer
    • Default value: 1
    • Default Value Role: By default warning is displayed
    • Values:
      Warning is displayed 1
      Warning is not displayed 0
  • PRT_Settings_SurfaceBased_Color
    Role: Setting to manage if the surface inherit colors from the input surface
    Detailed role: Color will inherit or not the surface color and can not be overloaded
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the color is inherited from input surface
    • Values:
      0 The color is not inherited from input surface
      1 The color is inherited from input surface
  • PRT_Settings_Thread
    Role: Setting to manage the display of the thread/tap in the 3D
    Detailed role: Displays a symbolic representation of the thread/tap.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the thread/tap is not displayed
    • Values:
      0 The thread/tap will not be displayed
      1 The thread/tap will be displayed
  • PRT_Settings_Thread_Standard
    Role: Setting to manage if the main diameter is used as reference for standard management
    Detailed role: Use or not use the main diameter as reference for standard management
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the main diameter is used as reference for standard management
    • Values:
      0 The main diameter not used as reference for standard management
      1 The main diameter used as reference for standard management
  • PRT_Settings_Vid_Panel
    Role: Setting to manage dialog display
    Detailed role: Dialog is displayed with new appearance
    • Type: integer
    • Default value: 0
    • Default Value Role: By default Dialog is displayed with new appearance
    • Values:
      1 Dialog is displayed with old appearance
      0 Dialog is displayed with new appearance
