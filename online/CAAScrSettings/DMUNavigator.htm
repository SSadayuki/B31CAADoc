DMUNavigator Setting Repository
Quick Reference

DMUNavigator

Role: Setting to handle the DMU navigator parameters

The DMUNavigator repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/DMU Navigator
  • Repository File: 4DNavigator
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DMUNavigator setting repository attributes:

  • ClearanceVoxel
    Role: clearance value
    • Type: float
    • Default value: 0.0
  • DMUReviewName
    Role: name of dmureview
    • Type: string
    • Default value: DMU Review
  • ForceClearanceVoxel
    Role: force clearance value in command
    • Type: integer
    • Default value: 0
  • ForceVoxel
    Role: force voxel
    • Type: integer
    • Default value: 0
  • MarkerAutoUpdate
    Role: update on product structure modification
    • Type: integer
    • Default value: 1
  • NumUrlName
    Role: name of hyperlink
    • Type: integer
    • Default value: 1
  • PublishAutoLaunchBrowser
    Role: open of browser for publish command
    • Type: integer
    • Default value: 1
  • SceneDefaultsColor
    Role: scene background color
    • Type: unsigned integer
    • Default value: 100
    • Default Value Role: RGB values. Value should be in [0..255]
    • Min value: 0
    • Max value: 255
