CAT2DLToolsOptions

Role: Manages 2D Layout for 2D Design options

The CAT2DLToolsOptions repository has the following characteristics:

  • Primary Tab Page: /Mechanical Design/2D Layout for 3D Design/View Creation

  • Full Compatibility: 1
  • Upward Compatibility: 1
