RenderingSettingsOutput Setting Repository
Quick Reference

RenderingSettingsOutput

Role: Photo Studio options

Detailed role: Full explanation of the Setting Role

The RenderingSettingsOutput repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/Photo Studio/Output
  • Repository File: RenderingOptionsSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

RenderingSettingsOutput setting repository attributes:

  • OutputHeight
    Role: Set output image width for quick render
    • Type: integer
    • Default value: 300
  • OutputPath
    Role: Set output image path for quick render
    • Type: string
    • Default value:
  • OutputSizeFrom
    Role: Set output image size for quick render
    • Type: integer
    • Default value: 1
    • Values:
      1 Size from active viewpoint
      2 Fixed size
  • OutputType
    Role: Set output image location for quick render
    • Type: integer
    • Default value: 1
    • Values:
      1 On screen
      2 On disk
  • OutputWidth
    Role: Set output image width for quick render
    • Type: integer
    • Default value: 400
  • SaveAutoIncrement
    Role: Enable automatic increment of image name
    • Type: boolean
    • Default value: false
