CATReconcileSettings Setting Repository
Quick Reference

CATReconcileSettings

Role: Reconciliator settings

Detailed role: Setting to handle Reconciliator parameters

The CATReconcileSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/ProductStructure/Reconciliation
  • Repository File: CATReconcileSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: CATPDMReconcile
    • Interface_CPP: CATIScmReconcileSettingsCtrl
    • LateType: CATEScmReconcileSettingsCtrl

CATReconcileSettings setting repository attributes:

  • AuthorizeLockAutomaticAcquisition
    Role: Authorize automtic lock acquisition.
    • Type: boolean
    • Default value: true
  • Environment
    Role: Environment
    • Type: string
    • Default value: ENOVIA5
    • Values:
      ENOVIA5 ENOVIA V5 VPM
      VPM1 ENOVIA VPM V4
      TeamPDM SmarTeam
      DC5PDM DesignerCentral PDM
  • InputXSL
    Role: Enable to define XSL steel sheet to be used for Reconciliator reporting solution.
    • Type: URN
    • Default value:
  • InputXSLPathType
    Role: Enable to define if XSL path is a logical or physical file.
    • Type: integer
    • Default value: 0
    • Values:
      0 XSL Path is a physical path.
      1 XSL Path is a logical path.
  • KeepExtRequestAuthorizationPublicationExposed
    Role: Enable to authorize Keep as external rule for Publication exposed
    • Type: boolean
    • Default value: true
  • LoadRequestAuthorizationPublicationExposed
    Role: Enable to authorize reload from VPDM rule for Publication exposed
    • Type: boolean
    • Default value: true
  • NbChar
    Role: Number of characters taken into account for the query:
    • Type: integer
    • Default value: 20
    • Min value: 1
    • Max value: 128
    • Step: 1
  • NewFromRequestAuthorizationPublicationExposed
    Role: Enable to authorize New From rule for Publication exposed
    • Type: boolean
    • Default value: false
  • NewRequestAuthorizationPublicationExposed
    Role: Enable to authorize New rule for Publication exposed
    • Type: boolean
    • Default value: true
  • NewRequestAuthorizationStructureExposed
    Role: Enable to authorize New rule for Structure exposed
    • Type: boolean
    • Default value: true
  • NewRevisionRequestAuthorizationPublicationExposed
    Role: Enable to authorize New Revision rule for Publication exposed
    • Type: boolean
    • Default value: true
  • NewVersionByDeltaRequestAuthorizationStructureExposed
    Role: Enable to authorize New Version by delta rule for Structure exposed
    • Type: boolean
    • Default value: true
  • NewVersionRequestAuthorizationPublicationExposed
    Role: Enable to authorize New Version rule for Publication exposed
    • Type: boolean
    • Default value: true
  • NewVersionRequestAuthorizationStructureExposed
    Role: Enable to authorize New version rule for Structure exposed
    • Type: boolean
    • Default value: true
  • OutputDirectory
    Role: Enable to define default location where to save Reconciliator reports.
    • Type: string
    • Default value: UNIX: /tmp Windows: ${CATTemp}
  • OverwriteByDeltaRequestAuthorizationStructureExposed
    Role: Enable to authorize Overwrite by delta rule for Structure exposed
    • Type: boolean
    • Default value: true
  • OWriteRequestAuthorizationPublicationExposed
    Role: Enable to authorize Overwrite rule for Publication exposed
    • Type: boolean
    • Default value: true
  • OWriteRequestAuthorizationStructureExposed
    Role: Enable to authorize Overwrite rule for Structure exposed
    • Type: boolean
    • Default value: true
  • ReloadRequestAuthorizationStructureExposed
    Role: Enable to authorize Reload from VPDM rule for Structure exposed
    • Type: boolean
    • Default value: true
