DMUMarker Setting Repository
Quick Reference

DMUMarker

Role: Settings related to DMU markers

The DMUMarker repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/DMU Marker
  • Repository File: 4DNavigator
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
