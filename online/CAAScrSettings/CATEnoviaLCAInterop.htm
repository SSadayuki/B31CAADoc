CATEnoviaLCAInterop Setting Repository
Quick Reference

CATEnoviaLCAInterop

Role: Setting to handle the ENOVIA V5 Settings

Detailed role: Setting to handle the ENOVIA V5 Settings

The CATEnoviaLCAInterop repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/ENOVIA V5 VPM
  • Repository File: CATEnoviaLCAInterop
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATEnoviaLCAInterop setting repository attributes:

  • AttrMapCheckBox
    Role: Work with Attribute Mapping
    • Type: boolean
    • Default value: false
  • AttrMapFile
    Role: Attribute Mapping File.
    • Type: string
    • Default value:
  • BackgroundColor
    Role: PVR Background color
    • Type: unsigned integer
    • Default value: 207
    • Default Value Role: Gray in RGB order
    • Min value: 0
    • Max value: 255
  • BuildAppExt
    Role: Working in visualization mode, the product applicative facets are not built: open is quicker
    • Type: boolean
    • Default value: true
  • ComputeAll
    Role: All documents in session will be analyzed to determine their refreshable status
    • Type: integer
    • Default value: 1
  • DeactivateDefaultMapping
    Role: De-activate Default Mapping
    • Type: boolean
    • Default value: false
  • DefaultLockOption
    Role: Part identifier attribute
    • Type: string
    • Default value: RetainLocks
  • DMUMode
    Role: View Product Structure Only / Work in DMU Mode
    Detailed role: All non-necessary properties for DMU (constraints, connectors, contextual design ...) will not be transferred during the whole CATIA session /
    All non-necessary properties are not transfered
    • Type: boolean
    • Default value: false
  • DocName_Attribute
    Role: Document identifier attribute
    • Type: string
    • Default value: V_ID
  • ExcludeLineicDataBBox
    Role: Exclude two diemensional elements from Bounding Box computation
    • Type: boolean
    • Default value: false
  • KeepLastSaveModes
    Role: Keep the last save modes on the entities
    • Type: boolean
    • Default value: false
  • NoAutoDuplicateName
    Role: Disable the automatic renaming at save process. The save of document or reference whose name already exist in data base doesn't occur
    • Type: boolean
    • Default value: false
  • NoTrimOnSave
    Role: Allows to customize the save to ENOVIA V5 VPM
    • Type: boolean
    • Default value: false
  • PartName_Attribute
    Role: Part identifier attribute
    • Type: string
    • Default value: V_ID
  • Refresh Criterion
    Role: Filter refreshable documents on
    • Type: integer
    • Default value: 0
    • Values:
      -1 No Criteria
      0 Filter criteria based on modification date parameter in server
      1 Filter criteria based on a counter from the document which is updated during Save opertaion
  • StructExp
    Role: Insert New CATProduct in Session
    • Type: integer
    • Default value: 0
    • Values:
      0 Publications Exposed
      1 Structure Exposed
  • WarningDisplay
    Role: Display Warning message if publication exposed mode is forced
    • Type: boolean
    • Default value: false
