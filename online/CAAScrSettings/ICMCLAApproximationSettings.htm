ICMCLAApproximationSettings Setting Repository
Quick Reference

ICMCLAApproximationSettings

Role: Approximation default parameters for ISD commands with Approximation tab

Detailed role: For all ISD commands with an Approximation tab Approximation default parameters are provided to be used for filling in certain Approximation parameters on demand.

The ICMCLAApproximationSettings repository has the following characteristics:

  • Primary Tab Page: /Shape/ICEM Shape Design/Approximation
  • Repository File: ICMGlobalUserSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ICMCLAApproximationSettings setting repository attributes:

  • CurveG0
    Role: G0 tolerance for curve approximation
    • Type: double
    • Default value: 0.001
  • CurveOrderU
    Role: Order for curve approximation (according to order type)
    • Type: integer
    • Default value: 7
  • CurveOrderUType
    Role: Type of order for curve approximation: Max/Min/Fix
    • Type: integer
    • Default value: 0
  • CurveSegmentsU
    Role: Number of segments for curve approximation (according to segments type)
    • Type: integer
    • Default value: 1
  • CurveSegmentsUType
    Role: Type of segments for curve approximation: Max/Min/Fix
    • Type: integer
    • Default value: 1
  • SurfaceG0
    Role: G0 tolerance for surface approximation
    • Type: double
    • Default value: 0.001
  • SurfaceG1
    Role: G1 tolerance for surface approximation
    • Type: double
    • Default value: 0.0043633231
  • SurfaceOrderLinked
    Role: Flag if u- and v-order for surface approximation should be linked
    • Type: integer
    • Default value: 0
  • SurfaceOrderU
    Role: U-Order for surface approximation (according to u-order type)
    • Type: integer
    • Default value: 7
  • SurfaceOrderUType
    Role: Type of u-order for surface approximation: Max/Min/Fix
    • Type: integer
    • Default value: 0
  • SurfaceOrderV
    Role: V-Order for surface approximation (according to v-order type)
    • Type: integer
    • Default value: 7
  • SurfaceOrderVType
    Role: Type of v-order for surface approximation: Max/Min/Fix
    • Type: integer
    • Default value: 0
  • SurfaceSegmentsLinked
    Role: Flag if u- and v-segments for surface approximation should be linked
    • Type: integer
    • Default value: 0
  • SurfaceSegmentsU
    Role: U-Segments for surface approximation (according to u-segments type)
    • Type: integer
    • Default value: 1
  • SurfaceSegmentsUType
    Role: Type of u-segments for surface approximation: Max/Min/Fix
    • Type: integer
    • Default value: 1
  • SurfaceSegmentsV
    Role: V-Segments for surface approximation (according to v-segments type)
    • Type: integer
    • Default value: 1
  • SurfaceSegmentsVType
    Role: Type of v-segments for surface approximation: Max/Min/Fix
    • Type: integer
    • Default value: 1
