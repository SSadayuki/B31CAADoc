DNBVNCSimulSettings Setting Repository
Quick Reference

DNBVNCSimulSettings

Role: Lists the settings used during the machine simulation in the Machining applications

Detailed role: The machining simulation settings are useful to manages options used during a machine simulation

The DNBVNCSimulSettings repository has the following characteristics:

  • Primary Tab Page: /Manufacturing/Simulation
  • Repository File: DNBVNCSimulSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBVNCSimulSettings setting repository attributes:

  • DisplayToolColor
    Role: Display tool color on stock
    Detailed role: Display tool color on stock
    • Type: boolean
    • Default value: true
  • DisplayTPInSimul
    Role: Defines if the tool path is shown during the machine simulation
    Detailed role: Defines if the tool path is shown during the machine simulation
    • Type: boolean
    • Default value: false
  • DisplayTPPathInSimul
    Role: Defines if the lines in the tool path are shown during the machine simulation
    Detailed role: Defines if the lines in the tool path are shown during the machine simulation
    • Type: boolean
    • Default value: true
  • DisplayTPPointInSimul
    Role: Defines if the points in the tool path are shown during the machine simulation
    Detailed role: Defines if the points in the tool path are shown during the machine simulation
    • Type: boolean
    • Default value: true
  • ISOPanelDisplay
    Role: ISOPanelDisplay
    Detailed role: ISOPanelDisplay
    • Type: boolean
    • Default value: false
  • NCCodeSimulMode
    Role: Defines if material removal path is done during the machine simulation
    Detailed role: Defines if material removal path is done during the machine simulation
    • Type: boolean
    • Default value: false
  • PaddingRange
    Role: Defines the number of segments on either side of the collision point
    Detailed role: Defines the number of segments on either side of the collision point
    • Type: integer
    • Default value: 5
  • TCPLineColor
    Role: Defines the lines color of the tool path during the machine simulation
    Detailed role: Defines the lines color of the tool path during the machine simulation
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: White in RGB order
    • Min value: 0
    • Max value: 255
  • TCPPathThickness
    Role: Defines the lines thickness of the tool path during the machine simulation
    Detailed role: Defines the lines thickness of the tool path during the machine simulation
    • Type: unsigned integer
    • Default value: 1
    • Default Value Role: Line 0.13mm
    • Min value: 1
    • Max value: 55
  • TCPPathType
    Role: Defines the lines type of the tool path during the machine simulation
    Detailed role: Defines the lines type of the tool path during the machine simulation
    • Type: unsigned integer
    • Default value: 1
    • Default Value Role: Line
    • Min value: 1
    • Max value: 7
  • TCPPointColor
    Role: Defines the points color of the tool path during the machine simulation
    Detailed role: Defines the points color of the tool path during the machine simulation
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Red in RGB order
    • Min value: 0
    • Max value: 255
  • TCPPointType
    Role: Defines the points type of the tool path during the machine simulation
    Detailed role: Defines the points type of the tool path during the machine simulation
    • Type: unsigned integer
    • Default value: 9
    • Default Value Role: Point
    • Min value: 1
    • Max value: 9
