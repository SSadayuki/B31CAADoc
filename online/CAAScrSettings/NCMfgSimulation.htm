NCMfgSimulation Setting Repository
Quick Reference

NCMfgSimulation

Role: Lists the settings of the resources used in the Machining applications

Detailed role: The resources machining settings are useful to manages options on tools and ressources catalogs

The NCMfgSimulation repository has the following characteristics:

  • Primary Tab Page: /Machining/Simulation
  • Repository File: NCMfgSimulation
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

NCMfgSimulation setting repository attributes:

  • BoxType
    Role: Fault box
    Detailed role: Display of bounding box
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the fault will be displayed as wireframe
    • Values:
      0 Wireframe
      1 Transparent
      2 None
  • CollisionVideo
    Role: Collision detection
    Detailed role: Action to be taken when collision is detected
    • Type: integer
    • Default value: 0
    • Default Value Role: By default collision will be Ignored
    • Values:
      0 Ignores the collision
      1 Stop when collision
      2 Continue when collision
  • ComputeAllInformationAtPickedPoint

      0 Smaller

    • Min value: 0.0
