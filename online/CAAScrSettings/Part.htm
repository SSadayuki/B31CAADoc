Part Setting Repository
Quick Reference

Part

Role: Setting to manage the options of PartDesign

Detailed role: PartDesign setting contains options necessary to the commands of PartDesign

The Part repository has the following characteristics:

  • Primary Tab Page: /Mechanical Design/Part Design/Part Design
  • Repository File: Part
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

Part setting repository attributes:

  • PRT_Settings_NonManifold_Warning
    Role: Setting to manage the display of warning message for Non-manifold input
    Detailed role: Warning will be displayed
    • Type: boolean
    • Default value: true
  • PRT_Settings_SurfaceBased_Color
    Role: Setting to manage if the surface inherit colors from the input surface
    Detailed role: Color will inherit or not the surface color and can not be overloaded
    • Type: boolean
    • Default value: true
  • PRT_Settings_Technological_Result_Parms
    Role: Element single segment mode
    Detailed role: Only single segment element will be created
    • Type: boolean
    • Default value: false
  • PRT_Settings_Thread
    Role: Setting to manage the display of the thread/tap in the 3D
    Detailed role: Displays a symbolic representation of the thread/tap.
    • Type: boolean
    • Default value: false
  • PRT_Settings_Thread_Standard
    Role: Setting to manage if the main diameter is used as reference for standard management
    Detailed role: Use or not use the main diameter as reference for standard management
    • Type: boolean
    • Default value: true
  • PRT_Settings_Vid_Panel
    Role: Setting to manage dialog display
    Detailed role: New appearance dialog will be displayed
    • Type: boolean
    • Default value: false
