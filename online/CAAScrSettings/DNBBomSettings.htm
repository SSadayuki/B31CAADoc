DNBBomSettings Setting Repository
Quick Reference

DNBBomSettings

Role: Tab page to handle the MBOM Setting Parameters

The DNBBomSettings repository has the following characteristics:

  • Primary Tab Page: /DPManufacturing/MBOM
  • Repository File: DNBBomSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: DPMSettingsInterfaces
    • Framework_IDL: DPMSettingsInterfaces
    • Interface_CPP: DNBIDSIMBOMViewerSettingAtt
    • Interface_IDL: DNBIAMBOMViewerSettingAtt
    • LateType: DNBDSIMBOMViewerSettingCtrl
