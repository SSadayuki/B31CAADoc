ShapeDesignGeneral Setting Repository
Quick Reference

ShapeDesignGeneral

Role: Short Qualification of the usage of the setting

Detailed role: Full explanation of the Setting Role

The ShapeDesignGeneral repository has the following characteristics:

  • Primary Tab Page: /Shape/Generative Shape Design/General
  • Repository File: ShapeDesign
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ShapeDesignGeneral setting repository attributes:

  • AxisLineVisuOption
    Role: Axis visualization limited to the bounding box
    • Type: short integer
    • Default value: 1
    • Values:
      0 false
      1 true
  • CtxMenuOnOneLevel
    Role: Contextual menu on one level
    • Type: short integer
    • Default value: 0
    • Values:
      0 false
      1 true
  • GroupInputOption
    Role: Integration of created features as group inputs
    • Type: short integer
    • Default value: 0
    • Values:
      0 false
      1 true
  • GSMTempAnalysisDefaultBehavior
    Role: Stacked analysis default behavior set as temporary
    • Type: boolean
    • Default value: false
  • InheritGPMode
    Role: Setting to manage if the feature inherits graphic properties from the input
    • Type: short integer
    • Default value: 0
    • Values:
      0 false
      1 true
  • PRT_Settings_Vid_Panel
    Role: Setting to manage dialog display
    Detailed role: New appearance dialog will be displayed
    • Tab Page: /Mechanical/Part Design
    • Type: boolean
    • Default value: false
  • TMDeviation
    Role: Maximum deviation
    • Type: double
    • Default value: 0.001
  • TMMergingDistance
    Role: Merging distance
    Detailed role: Merging distance
    • Type: double
    • Default value: 0.001
  • TMSmoothingType
    Role: Continuity type
    • Type: integer
    • Default value: 0
    • Values:
      0 None
      1 Tangency
      2 Curvature
  • TMTolerantLayDownState
    Role: Toleranl laydown state
    • Type: boolean
    • Default value: false
