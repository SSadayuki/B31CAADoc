Replan Setting Repository
Quick Reference

Replan

Role: Settings to handle the Automatic Re-plan of ERs settings

The Replan repository has the following characteristics:

  • Primary Tab Page: /DPManufacturing/Re-plan
  • Repository File: Replan
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

Replan setting repository attributes:

  • ExtendedEffectivity
    Role: Allows to define Extended Effectivity
    Detailed role: Allows to define Extended Effectivity on New Relations
    • Alias: ExtendedEffectivity
    • Type: string
    • Default value:
  • OldRelations
    Role: Defines the type of Old Relation
    Detailed role: Defines the type of Old Relation 1.Cut Effectivity 2.Delete 3.Unchanged
    • Alias: OldRelations
    • Type: integer
    • Default value: 1
  • ScopeOfReplan
    Role: Defines the Scope of Replan
    Detailed role: Defines the Scope of Replan 1.All in Project 2.All in Session
    • Alias: ScopeOfReplan
    • Type: integer
    • Default value: 1
