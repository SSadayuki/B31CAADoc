CCDCompatibilitySettings Setting Repository
Quick Reference

CCDCompatibilitySettings

Role: Settings that control the import and export of CCD drawings.

Detailed role: Settings that control the import and export of CCD drawings.

The CCDCompatibilitySettings repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/CCD
  • Repository File: CCDCompatibilitySettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CCDCompatibilitySettings setting repository attributes:

  • ExportConvertToLineWeights
    Role: V5 line widths are converted to CCD line weights using CCD Style1 mappings.
    • Type: boolean
    • Default value: true
  • ExportSplineOption
    Role: If true, V5 curves are converted to CCD polylines using a linear approximation.
    • Type: boolean
    • Default value: false
  • ExportSplineSagValue
    Role: The sag value in MKS defines the chordal deviation used in the linear approximation.
    • Type: double
    • Default value: 0.00001
  • HeavyLineWeight
    Role: Mapping value for CCD heavy line weight
    • Type: integer
    • Default value: 3
    • Min value: 1
    • Max value: 55
  • ImportDftStandard
    Role: Name of the Drafting standard that defines the standard of the resulting Drawing
    • Type: string
    • Default value: ISO_DS
  • ImportDimensionExplode
    Role: If true, dimension objects from CADAM models will be converted into V5 drawing elements.
    • Type: boolean
    • Default value: false
  • ImportSplineOption
    Role: If true, CADAM Splines are converted to V5 polylines using a linear approximation.
    • Type: boolean
    • Default value: false
  • ImportSplineSagValue
    Role: The sag value in MKS defines the chordal deviation used in the linear approximation.
    • Type: double
    • Default value: 0.00001
  • ImportSymCatPath
    Role: Directory of CADAM Symbol Tables translated into CATIA V5 Catalogs.
    • Type: string
    • Default value:
  • LightLineWeight
    Role: Mapping value for CCD light line weight
    • Type: integer
    • Default value: 1
    • Min value: 1
    • Max value: 55
  • MediumLineWeight
    Role: Mapping value for CCD medium line weight
    • Type: integer
    • Default value: 2
    • Min value: 1
    • Max value: 55
  • Style1Break
    Role: V5 line type mapping to CCD style 1 break line type
    • Type: integer
    • Default value: 8
    • Min value: 1
    • Max value: 63
  • Style1Center
    Role: V5 line type mapping to CCD style 1 center line type
    • Type: integer
    • Default value: 4
    • Min value: 1
    • Max value: 63
  • Style1Dash
    Role: V5 line type mapping to CCD style 1 dash line type
    • Type: integer
    • Default value: 2
    • Min value: 1
    • Max value: 63
  • Style1Phantom
    Role: V5 line type mapping to CCD style 1 phantom line type
    • Type: integer
    • Default value: 5
    • Min value: 1
    • Max value: 63
  • Style1Solid
    Role: V5 line type mapping to CCD style 1 solid line type
    • Type: integer
    • Default value: 1
    • Min value: 1
    • Max value: 63
  • Style2LineTypes
    Role: V5 line type mappings to CCD style 2 line types in an array
    • Type: integer
    • Default value: 6
    • Min value: 1
    • Max value: 63
