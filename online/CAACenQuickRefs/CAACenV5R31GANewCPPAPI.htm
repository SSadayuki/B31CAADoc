New C++ Authorized APIs in CAA V5-6R2021 GA

The following are the new CAA V5-6R2021 GA C++ Authorized APIs, compared with CAA V5-6R2020 at GA level.

  • CATTPSInterfaces framework
    • Interface CATITPSEdgeOfUndefinedShape
  • DraftingInterfaces framework
    • Enumeration DftBlankingMode
    • Enumeration DftLineSpacingDefinition
    • Enumeration DftMirroringDefinition
    • Enumeration DftPitchDefinition
    • Enumeration DftScaleMode
    • Enumeration DftUnitDisplayMode
  • GMOperatorsInterfaces framework
    • Macro or #define CATICGMTopWireContinuity_V1
    • Global Function CATCGMCreateTopSilhouette
  • GeometricObjects framework
    • Macro or #define CATJointChamferType
