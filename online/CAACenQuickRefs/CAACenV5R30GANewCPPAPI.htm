New C++ Authorized APIs in CAA V5-6R2020 GA

The following are the new CAA V5-6R2020 GA C++ Authorized APIs, compared with CAA V5-6R2019 at GA level.

  • CATGSMUseItf framework
    • Enumeration CATGSMMidSurfaceComputationMethod
    • Enumeration CATGSMMidSurfaceComputationMode
    • Enumeration CATGSMMidSurfaceCreationMode
    • Interface CATIGSMUseMidSurface
  • CATTPSInterfaces framework
    • Interface CATITPSPrincipalFeature
    • Interface CATITPSRepresentationStrings
  • GMOperatorsInterfaces framework
    • Enumeration CATFrFCuttingOption
    • Enumeration CATFrFInflexionTreatment
    • Enumeration CATFrFSmoothingParamType
    • Enumeration CATFrFTestOfConvergence
  • GSMInterfaces framework
    • Interface CATIGSMMidSurface
  • Mathematics framework
    • Class CATLISTP(CATMathBox)
  • ObjectModelerCollection framework
    • Macro or #define E_KEYNOTFOUND
    • Macro or #define E_NOLOCALVALUE
    • Macro or #define E_NOTALIST
    • Macro or #define E_OUTOFBOUND
    • Macro or #define E_TYPEMISMATCH
  • ObjectSpecsLegacy framework
    • Macro or #define E_AGGBOUNDARYERROR
    • Macro or #define E_AGGMULTIPLEOWNERERROR
    • Macro or #define E_INVALIDFEATURE
  • ProductStructure framework
    • Interface CATPrsContextualLinkServices
