Updating a Workspace Unix Image

This task shows you how to update the Unix workspace you have set as an image of your current Windows workspace using the Unix bridge after you brought changes to this workspace.
Before you begin: Note that:
  • The DS Unix bridge must be installed. See Installing C++ Interactive Dashboard.
  • The DS Unix bridge properties must first be set. See Accessing and Customizing Settings.
  • The Windows workspace used to update the Unix image must be opened in the 3DS Workspace Explorer.

  1. Open the Windows workspace in the 3DS Workspace Explorer.

    See Opening a Workspace.

  2. On the Tools menu, click Update WS on Unix.

    The Connect dialog box opens.

     

  3. In the User Name box, type your Unix user name.

  4. In the Password box, type your Unix user password.

  5. Click OK.

    The Unix image is updated with respect to the Windows workspace contents.

Warning: Your password is never stored on disk. It is encrypted and kept in memory It sis deleted from memory when you exit Visual Studio.
