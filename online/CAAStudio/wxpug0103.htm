Declaring a Unix Image of a Windows Workspace

This task shows you how to declare a Unix image workspace of a Windows workspace to a Unix computer using the Unix bridge.

Before you begin: Note that:
  • The DS Unix bridge must be installed. See Installing C++ Interactive Dashboard.
  • The DS Unix bridge settings must first be set. See Accessing and Customizing Settings.
  • The Windows workspace used to create the Unix image must be opened in the 3DS Workspace Explorer.

  1. Open the Windows workspace in the 3DS Workspace Explorer.

    See Opening a Workspace.

  2. On the Tools menu, click Unix Bridge properties.

    The Bridge Properties dialog box opens.

     

  3. In the UNIX Workspace path box, type the path of the directory that will contain the image workspace of the Windows workspace.

  4. In the UNIX Workspace prerequisites box, type the path of the Unix workspace containing the prerequisites of the image workspace to create.

    If several workspaces are needed, separate their paths with semicolons.

  5. Click OK.

    The Unix workspace is declared as an image of the Windows one.
