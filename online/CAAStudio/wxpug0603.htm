Running Commands on Unix

This command enables you to run Unix commands in the Unix computer hosting the Unix image of your Windows workspace.
Before you begin:
  • The DS Unix bridge must be installed. See Installing C++ Interactive Dashboard.
  • The DS Unix bridge settings must first be set. See Accessing and Customizing Settings.
  • The Windows workspace for which you want to run commands in its Unix image must be opened in the 3DS Workspace Explorer. See Opening a Workspace.
  • The Unix image must be built and its run time view refreshed. See Compiling on Unix and Refreshing the Unix Run Time View.

  1. On the Tools menu, click Execute on Unix.

    The Connect dialog box opens if you were not connected to Unix.

  2. In this case, in the User Name box, type your Unix user name.

  3. In the Password box, type your Unix user password.

  4. Click OK.

    The Execute on Unix dialog box opens.

  5. In the Command to execute list, type or select the name of the command to execute, such as cd, ls, ps -ef, ok mkmk.

  6. If you want to run a command such as mkmk or mkrun that requires to use the installed RADE build environment in the Unix computer, check Use Build environment.

  7. Click Execute.

    The command is executed.

Warning: Your password is never stored on disk. It is encrypted and kept in memory It is deleted from memory when you exit Visual Studio.
