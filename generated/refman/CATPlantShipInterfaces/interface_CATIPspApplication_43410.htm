CATPlantShipInterfaces Framework: CATIPspApplication Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIPspApplication

Interface to initialize and query an application.
Role: To activate and query a Plant Ship application.


Method Index
GetApplicationDomain(CATString&) Retrieve the application main domain.
Initialization(CATIContainer*) Initialize the application environment (load feature start up objects, activate the application.
Is3DApplication(CATBoolean*) Is the application that we are dealing with a 3D Application?.
ListDomains(CATICStringList**) Retrieve a list of all domain names supported by this application.
LoadFeatFiles(CATIContainer*) Load the associated application domains *.
