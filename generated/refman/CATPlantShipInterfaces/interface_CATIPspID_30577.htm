CATPlantShipInterfaces Framework: CATIPspID Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIPspID

Interface to generate and set IDs for the Plant Ship Objects.
Role: This is the interface for Plant Ship object ID generation.


Method Index
GenAndPutID(CATUnicodeString&,int) ID is generated with the sequence number and is stored on the object.
GenAndPutIDNoGenSeqNum(CATUnicodeString&) ID is generated without sequence number and is stored on the object.
GenIDNoGenSeqNum(CATUnicodeString&) ID is generated without sequence number.
GetID(CATUnicodeString&) Get ID of the object.
GetLocalID(CATUnicodeString&) Get LocalID of the object.
IsIDGenerated(CATBoolean&) Is Object ID generated by the ID schema.
SetID(CATUnicodeString&) Set ID of the object with the user defined string.
