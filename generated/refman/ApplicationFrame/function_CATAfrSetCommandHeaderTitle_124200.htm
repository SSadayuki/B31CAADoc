CATAfrSetCommandHeaderTitle( const iCommandHeaderId, const iTitle)
Changes the icon of a command header.
Parameters:
iCommandHeaderId
The identifier of the command header.
iTitle
The unicode string containing the title.
Returns:
S_OK
the service has succeeded and the title was changed.
E_FAIL
the service has failed and the title was not changed.
