ApplicationFrame Framework:SetAccessNext Macro

Macro SetAccessNext (variableName,nextName)
Chains an access to a previous one in a container.
Role: Container's accesses are chained. The first access is attached to its container by the macro. To chain accesses you use this macro.
Parameters:
variableName
The pointer to an access instance already chained by a SetAccessNext or SetAccessChild macro.
nextName
The pointer to the access instance which follows variableName
