ApplicationFrame Framework: CATExtInPlaceSite

Usage: you can derive this class.

public class CATExtInPlaceSite

Adapter for the CATInPlaceSite interface.
Role:To implement the interface CATInPlaceSite for a class MyClass, write a class MyClassExtInPlaceSite subclassed from CATExtInPlaceSite and defined as an data extension of MyClass. MyClassExtInPlaceSite may override various methods, but in most case there is nothing to do.
