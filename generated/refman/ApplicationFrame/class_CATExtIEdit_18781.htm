ApplicationFrame Framework: CATExtIEdit

Usage: you can derive this class.

public class CATExtIEdit

Adapter for the CATIEdit interface .
Role : This base class provides usefull services making the management of the life cycle of the command returned by the method easier.
The CATExtIEdit class provides the method that tells whether the instance returned by the method is in the active state or not.
Implementing CATIEdit just consists in subclassing the CATExtIEdit class and simply overriden the or the or both methods. Do not forget to implement the CATIEdit interface using : CATImplementBOA(CATIEdit, CATExtIEdit).
