CATAfrStartWorkbench( const iWorkbenchId)
Starts a workbench.
Role: This method starts a workbench as if the user had chosen it in the Start menu.
Parameters:
iWorkbenchId
The identifier of the workbench to start.
Returns:
S_OK
if the workbench was found and started
E_FAIL
if the workbench was not found
