ApplicationFrame Framework: CATCmdContainer

Usage: you must use this class as is. You should never derive it.

public class CATCmdContainer

Class to create the container's access.
Role: A container can be :

  • a menubar, a menu, a submenu, a toolbar or an icon box. Use the macro with CATCmdContainer as classname parameter to create an instance.
  • a workshop. Use the macro with CATCmdWorkshop as classname parameter to create an instance.
  • a workbench. Use the macro with CATCmdWorkbench as classname parameter to create an instance.

public AddChild( newItem, const beforeItem= NULL)
Declares an access to add in a container. Role: Container's accesses are chained. The access is attached to its container at the end of the container's children list.
