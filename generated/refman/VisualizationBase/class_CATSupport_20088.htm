Parameters:

iPoint
array of 3xiNbPlanes of coordinates. Each (x,y,z) represents a point belonging to a clipping plane. And there is only one point per plane.
iNnormal
array of 3xiNbPlanes of coordinates. Each (x,y,z) represents a normal to one plane. And there is only one normal per plane.

    public virtual GrabPixelImage( x, y, width, height, img= NULL, buffer= 0)
    Returns the grabbed CATPixelImage by reading the pixel rectangle at specified location (lower left corner) and size (rectangular block of pixels) from the framebuffer.
    Parameters:
    x
    Specify the X window coordinate of the first pixel that is read from the frame buffer.
    y
    Specify the Y window coordinate of the first pixel that is read from the frame buffer.
    width
    The width of the captured image that specifies the horizontal dimension of the pixel rectangle to read. width <= 0 is invalid.
    height
    The height of the captured image that specifies the vertical dimension of the pixel rectangle to read. height <= 0 is invalid.
    img
    The input pointer to the CATPixelImage. The default value for this parameter is NULL. Only CATPixelImageFormat RGBA is supported. If provided the pixels of img will be modified with pixel rectangle and the same is returned.
    buffer
    The frame buffer to read from. The default value for this parameter is 0. 0 = Front Buffer 1 = Back Buffer
    Returns:
    The pointer to the new or modifed input CATPixelImage. Values for pixels that lie outside the Buffer are undefined.
