VisualizationBase Framework: CAT2DCustomRep

Usage: you must use this class as is. You should never derive it.

public class CAT2DCustomRep

Class to manage a collection of 2D graphic primitives with different graphic attributes.
Role: The primitives that are being added to the representation are not copied within it. The graphical attributes are copied inside the representation.
