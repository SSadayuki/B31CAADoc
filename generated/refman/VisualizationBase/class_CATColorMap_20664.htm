VisualizationBase Framework: CATColorMap

Usage: you must use this class as is. You should never derive it.

public class CATColorMap

Class to create a table of 256 indexed colors.
Role: This class permits to have a relationship between an indexed color and its red, green and blue components.
The table index ranges from 0 to 255.
