DialogEngine Framework:ActionMethod Typedef

typedef CATBoolean (CATCommand::*ActionMethod)(void * iData)
Prototype of the action methods.
Role: These methods are passed as parameters of the class methods.
Values:
iData
The data specified by the method.
Returns:
The CATBoolean value used to combine actions. In an OrAction, the second action is executed only if the first action returns TRUE.
This object is included in the file: CATDiaAction.h
