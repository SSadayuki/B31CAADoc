CATAnalysisResources Framework: CATESamEnumManager

Usage: you can derive this class.

public class CATESamEnumManager

Adaptor for implementing CATISamEnumManager interface.
This inferface is to be implemented on a basic component build by derivation of SAMListManager.

DataExtension Implementation is mandatory.
