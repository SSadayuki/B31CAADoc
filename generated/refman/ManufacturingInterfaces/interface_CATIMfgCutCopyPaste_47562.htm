Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIMfgCutCopyPaste

Interface dedicated to manage Cut, Copy and Paste.
Role: This interface offers services to customize the cut, copy and paste mechanisms.
To define specific behaviours to your manufacturing object during the Cut/Copy/Paste, derive from the supplied ObjectCCP_SPEC adapter class and called in the implementation of each method of CATICutAndPastable the corresponding method of this interface. Thus, you retrieve the default mechanism provided by the manufacturing infrastructure.
