ManufacturingInterfaces Framework: CATIMfgNavigateFeatureView Interface

Usage: you can freely reimplement this interface.

interface CATIMfgNavigateFeatureView

Interface to display the manufacturing model in the Manufacturing View when sorted by features.
Role : This interface is to implemented by manufacturing features to customize the way they are displayed (label and children nodes) in the Manufacturing View sorted by features.



This object is included in the file: CATIMfgNavigateFeatureView.h
If needed, your Imakefile.mk should include the module: MfgItfEnv
