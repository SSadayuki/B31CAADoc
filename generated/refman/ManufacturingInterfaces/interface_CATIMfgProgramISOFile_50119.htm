ManufacturingInterfaces Framework: CATIMfgProgramISOFile Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIMfgProgramISOFile

Interface implemented on the Manufacturing Program.
The rule of this interface is to manage access to ISO file for simulation.


Method Index
CloseSession() Closing of working session (to call at the end of the use of this interface).
GetISORecord(int,CATUnicodeString&,int) Retrieves an ISO record in the ISO file.
GetIndexforMO(CATBaseUnknown_var&,int&,int&) Retrieves the value of start line and end line in the ISO file for a Manufacturing Operation.
GetNumberOfISORecords(int&,int) Retrieves the total number of ISO records.
InitSession() Initialization of working session (to call at the beginning of the use of this interface).
SetIndexforMO(CATBaseUnknown_var&,int,int) Sets an ISO record from the ISO file.
