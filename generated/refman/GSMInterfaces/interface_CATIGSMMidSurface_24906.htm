GSMInterfaces Framework: CATIGSMMidSurface Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIGSMMidSurface

Interface to Mid Surface feature.
Role: Allows you to gets or/and sets the definition elements of a Mid Surface.

See also:
