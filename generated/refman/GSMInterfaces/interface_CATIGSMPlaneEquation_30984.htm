GSMInterfaces Framework: CATIGSMPlaneEquation Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIGSMPlaneEquation

Interface to plane feature defined by an equation plane.
Role: Allows you to access data of the plane feature created by its cartesian equation. Plane equation is Ax+By+Cz = D.
