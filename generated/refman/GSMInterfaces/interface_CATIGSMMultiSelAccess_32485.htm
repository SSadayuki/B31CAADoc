GSMInterfaces Framework: CATIGSMMultiSelAccess Interface

Usage: you can freely reimplement this interface.

interface CATIGSMMultiSelAccess

Interface to access object reference in feature multisel.
Role: Allow to modify multisel parameters on the object
Note: this interface is implemented on every feature supporting multisel

See also:

public virtual CheckCoherency( iSpecMultiSel, iCoherent)
Checks consistancy.
This tool is used to verify consistancy with the aggregated feature regarding common reference elements All occurences of multisel feature should share the same common feature if aggregated ( ex:contextual feature)
Parameters:
iSpecMultisel
Multtisel object (Not used)
oCoherent
= TRUE OK = FALSE KO , some "normmally" commnon aggregated feature are detected diffetrents

Updates data related multisel features.
Notes :
1/ Lists are Input/ Output parameters The Lists are filled with the content of the multisel inputs of the current instance of the feature
