GSMInterfaces Framework: CATIGSMCircleCenterAxis Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIGSMCircleCenterAxis

Interface to circle feature defined by a center and an axis.
Role: Allows you to acess data of circle feature created from the center and axis/line specified.
