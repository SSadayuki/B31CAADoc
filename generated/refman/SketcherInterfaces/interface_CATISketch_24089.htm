SketcherInterfaces Framework: CATISketch Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATISketch

The methods of this interface provide the client with the functionality for handling a sketch.
Role:


Method Index
AttachToPlane(CATBaseUnknown_var&,int,int) Defines the planar support of the sketch (the sketch becomes Sliding).
GetAbsolute2DAxis(CATI2DAxis_var&) Returns the absolute axis of the sketch.
GetAxisData(double*,double*,double*) Returns the position of the sketch in space.
GetComponents(CATIdent,CATListValCATI2DWFGeometry_var&) Retrieves all geometries.
GetCurrentCenterLine() Returns the (unique) center line of the sketch.
GetPlanarSupport(CATISpecObject_var&) Returns the planar support of the sketch.
IsEmpty() Tells if the sketch is empty.
Isolate() Isolates the projected and intersected Geometries.
SetAxisData(double*,double*,double*) Defines the position of the sketch in space.
