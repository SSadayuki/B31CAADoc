SketcherInterfaces Framework: CATI2DWFFactory Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATI2DWFFactory

public virtual CreateChamfer( iCurve1, iCurve2, const iPoint, const iParam1=NULL, const iParam2=NULL, const iChamfer=1)
Creates a chamfer segment.
Parameters:
iCurve1
[in] First geometric element.
iCurve2
[in] Second geometric element.
iPoint
[in] Indicated 2D point coordinates.
iParam1
[in] Optional first parameter of the chamfer segment.
iParam2
[in] Optional second parameter of the chamfer segment.
iChamfer
[in] Indicates the computation mode of the chamfer, thus the significance of the two parameters iParam1 and iParam2: If iChamfer == 1 (Length of the chamfer and its angle): iParam1 means the length of the chamfer segment. iParam2 means the angle of the chamfer segment with the tangent to the first curve at the nearest intersection point from the indicated 2D point. If iChamfer == 2 (Length1 and Length2 define the chamfer): iParam1 means the distance of the chamfer segment with either the intersecting point of the two curves, if it exists, or the end point of the first curve. iParam2 means the distance of the chamfer segment with either the intersecting point of the two curves, if it exists, or the end point of the second curve. If iChamfer == 3 (Length1 and Angle define the chamfer): iParam1 means the distance of the chamfer segment with either the intersecting point of the two curves, if it exists, or the end point of the first curve. iParam2 means the angle of the chamfer segment with the tangent of the first curve at the nearest intersection point from the indicated 2D point.

public virtual CreateClothoid( iInitialCurve, const iInitialOrient=1, iInitialPoint=NULL, const iInitialSide=0, iFinalGeom=NULL, const iFinalOrient=0, const iParam1Type=0, const iParam1=NULL, const iParam2Type=0, const iParam2=NULL)
Creates a 2D clothoid curve.
Parameters:
iInitialCurve
[in] Tangent curve at the initial extremity of the clothoid curve.
iInitialOrient
[in] Optional: Orientation to be applied to the initial tangent.
Legal values:
1
same orientation
-1
opposite orientation
iInitialPoint
[in] Optional: Initial point. Useless when final extremity is defined with a circle.
iInitialSide
[in] Optional: Defines on which side of its initial oriented tangent the curve will turn. Useless when final extremity is defined with a circle or a point.
Legal values:
1
left side
0
undefined
-1
right side
iFinalGeom
[in] Optional: 2D Geometry defining totally or partially the final extremity of the clothoid curve. It may be null, or a point, a line, or a circle. If null, both Param1 and Param2 are required. A line specifies the tangent at the final extremity of the curve. Then Param1 is required and cannot be an angle. A point or a circle does not require any additional parameter.
iFinalOrient
[in] Optional: Orientation to be applied to the final tangent when iFinalGeom is a line.
Legal values:
1
same orientation
0
undefined
-1
opposite orientation
iParam1Type
[in] Optional: Specifies which kind of parameter is used as first parameter.
See also:
enum CAT2DClothoidParamType
Parameters:
iParam1
[in] Optional: Value of the first parameter.
iParam2Type
[in] Optional: Specifies which kind of parameter is used as second parameter.
See also:
enum CAT2DClothoidParamType
Parameters:
iParam2
[in] Optional: Value of the second parameter.
See also:
CATI2DClothoidCurve
