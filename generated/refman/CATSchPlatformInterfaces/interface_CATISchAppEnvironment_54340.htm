interface CATISchAppEnvironment

Interface to mangage application environment in the Schematic application context.
Role: To set up application environment (such as initializing workbenches, loading feature start up objects).


Method Index

AppGetAppObjectContainer(CATIContainer**) Return the container of the application objects.
