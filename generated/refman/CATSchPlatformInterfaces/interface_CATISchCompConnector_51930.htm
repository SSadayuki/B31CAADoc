CATSchPlatformInterfaces Framework: CATISchCompConnector Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATISchCompConnector

Interface to mangage Schematic object ability to make connections.
Role: To add or remove connectors.


Method Index
AddConnector(char*,CATISchGRRComp*,double*,CATISchAppConnector**) Add a connector.
AddDynamicConnector(char*,CATISchGRRComp*,double*,CATISchAppConnector**) Add a dynamic connector.
RemoveConnector(CATISchGRRComp*,CATISchAppConnector*) Remove a connector.

Methods


public virtual AddConnector( const iClassType, iGRR, iDb2CntrPosition, oNewSchCntr)
Add a connector.
Parameters:
iClassType
Class type of the connector to be added.
iGRR
The graphical image of the component to add the connector to. If NULL, connector will be added to all representations.
iDb2CntrPosition
The position of the connector (optional, it could be NULL).
oNewSchCntr
The new Schematic Connector object created.
Returns:
An HRESULT value.
Legal values:
S_OK
operation is successful
E_FAIL
operation failed
