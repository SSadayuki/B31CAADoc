CATSchPlatformInterfaces Framework: CATISchAppConnector Interface

Usage: you can freely reimplement this interface.

interface CATISchAppConnector

Interface to mangage application connector object behaviors.
Role: To specify application connector behaviors such as connect and disconnect.


Method Index
AppConnect(CATISchAppConnector*,CATISchAppConnection**) Connect to an input connector.
AppConnectBranch(CATISchAppConnector*,CATISchAppConnection**) Connect to an input connector for Branch.
AppDisconnect(CATISchAppConnector*) Disconnect from an input connector.
AppGetAssociatedConnectable(CATISchAppConnectable**) Find the application object that owns this connector.
AppIsCntrConnected(CATBoolean*) Query whether the connector has been connected.
AppListCompatibleTypes(CATICStringList**) Find all the class types of connector that are compatible with this connector for connections.
AppListConnections(CATICStringList*,CATIUnknownList**) Find all the connections that include this connector.
AppOKToNoShowConnectedCntr(CATBoolean*) Query whether it is OK to no show the connector after it is connected.
