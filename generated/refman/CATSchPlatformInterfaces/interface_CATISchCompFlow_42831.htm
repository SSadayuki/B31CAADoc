CATSchPlatformInterfaces Framework: CATISchCompFlow Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATISchCompFlow

Interface to manage Schematic component's association with its internal flow objects.
Role: To modify component's links to its internal flow objects.


Method Index
