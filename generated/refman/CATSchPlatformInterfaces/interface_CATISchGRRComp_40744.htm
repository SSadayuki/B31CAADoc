CATSchPlatformInterfaces Framework: CATISchGRRComp Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATISchGRRComp

Interface to manage the position and orientation of Schematic component graphic objects.
Role: To query or modify the position and orientation of a component graphic


Method Index
GetPosition(double*) Get the current position of the component.
GetRotationAngle(double*) Get the current rotation angle (from x-axis in radian) of the component.
GetScale(double*) Get the current scale factor of the component.
GetTransformation2D(double*) Get the local coordinate reference frame (with respect to absolute coordinate system) of the component.
SetPosition(double[2]) Set the current position of the component.
SetRotationAngle(double) Set the current rotation angle (from x-axis in radian) of the component.
SetScale(double) Set the current scale factor of the component.
SetTransformation2D(double[6]) Set the local coordinate reference frame (with respect to absolute coordinate system) of the component.
