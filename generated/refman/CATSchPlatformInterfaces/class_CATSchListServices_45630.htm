CATSchPlatformInterfaces Framework: CATSchListServices

Usage: you must use this class as is. You should never derive it.

public class CATSchListServices

Tools to manage list of pointers used in schematic platform interface.
Role: For creation and modification list of pointers.


Method Index

CreateCATIUnknownList(CATIUnknownList**) Create CATIUnknownList object for a list of IUnknown interface pointers.
