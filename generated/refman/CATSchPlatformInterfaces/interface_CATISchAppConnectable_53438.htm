CATSchPlatformInterfaces Framework: CATISchAppConnectable Interface

Usage: you can freely reimplement this interface.

interface CATISchAppConnectable

Interface to mangage application object behaviors in making connections.
Role: To specify application object behaviors such as adding a connector and removing a connector.


Method Index

public virtual AppListConnectables( iLCntbleClassFilter, oLCntbles, oLCntrsOnThisObj, oLCntrsOnConnected)
Find all the application objects connected to this object through their connectors.
Parameters:
oLCntrClassFilter
A list of all the class types for filtering the output application objects list.
oLCntbles
A list of application objects connected to this object. (members are CATISchAppConnectable interface pointers).
oLCntrsOnThisObj
A list of connectors on this object through which the connection is made. (members are CATISchAppConnector interface pointers).
oLCntrsOnConnected
A list of connectors on the connected objects through which the connection is made. (members are CATISchAppConnector interface pointers). Members in this list corresponds to those in oLCntrsOnThisObj in making the corresponding connections.

S_OK

operation failed
