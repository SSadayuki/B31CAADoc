CORBAServerInfra Framework: CATIUExitServerAdmin Interface

Usage: you can freely reimplement this interface.

interface CATIUExitServerAdmin

Interface to retrieve user information.
Role: This interface is a user exit. The server manager needs this information to run an applicative server for the user.
You should implement it on the CATUExitServerAdmin component by creating a data extension.
