CATPLMDictionary Framework: VPMIDicElement Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface VPMIDicElement

This class enables to manage all basics dictionary types role: Defines the dictionary basic type


Method Index
GetAlias(CATUnicodeString*) Retrieves the element alias
GetCategories(CATLISTP(IUnknown)*) Retrieves the categories to which the elment belongs.
GetExpositionLevel(CATUnicodeString*) Retrieves the Element's exposition level.
GetName(CATString*) Retrieves the dictionary object name.
GetUuid(CATDbBinary*) Retrieves the dictionary object UUID.
GetVisibility(VPMDicVISIBILITY*) Retrieves the visibility tag og the element
SetExpositionLevel(CATUnicodeString*) Sets the Element's exposition level.
