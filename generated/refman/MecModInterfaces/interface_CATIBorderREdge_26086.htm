MecModInterfaces Framework: CATIBorderREdge Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIBorderREdge

Interface to filter a selecting object of BorderREdge type.
Role: This interface allows to filter a selecting object ( ) of BorderREdge type. A BorderREdge is a border egde on a 2D topological object ( ). A BorderREdge has only one adjacent face.



This object is included in the file: CATIBorderREdge.h
If needed, your Imakefile.mk should include the module: CATMecModInterfaces
