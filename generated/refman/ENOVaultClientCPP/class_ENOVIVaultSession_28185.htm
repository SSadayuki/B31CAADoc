ENOVaultClientCPP Framework: ENOVIVaultSession

Usage: you must use this class as is. You should never derive it.

public class ENOVIVaultSession

Class to manage client vault session.
role: This class manages all the connections to the vaults It supplies methods to bind existing documents by URL and to get Objects.
