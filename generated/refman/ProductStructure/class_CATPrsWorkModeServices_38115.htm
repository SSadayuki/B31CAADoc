ProductStructure Framework: CATPrsWorkModeServices

Usage: you must use this class as is. You should never derive it.

public class CATPrsWorkModeServices

Class to retrieve and assign a work mode of/on a Product.
Role: The work mode is a loading mode which can vary according to the selective loading context.
