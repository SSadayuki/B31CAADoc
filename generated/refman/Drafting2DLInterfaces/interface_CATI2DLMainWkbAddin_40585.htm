Drafting2DLInterfaces Framework: CATI2DLMainWkbAddin Interface

Usage: you can freely reimplement this interface.

interface CATI2DLMainWkbAddin

Interface of 2D Layout For 3D Design workbench.
by implementing this interface you create an addin of Main workbench of Layout. The toolbars and the item menu will be only available when the workbench is the current one. CATIDrwAddin is the addin of the "Layout" working view workbench which appears in the "Mechanical Design" Solution.
