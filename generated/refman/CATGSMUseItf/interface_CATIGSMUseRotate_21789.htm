CATGSMUseItf Framework: CATIGSMUseRotate Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIGSMUseRotate

Rotate feature.
Role: Allows to access data of the Rotate feature created by using an element, an axis (a line) and an angle.
