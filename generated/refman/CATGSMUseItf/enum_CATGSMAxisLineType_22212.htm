enum CATGSMAxisLineType {
  CATGSMAxisLineType_None,
  CATGSMAxisLineType_MajorAxis,
  CATGSMAxisLineType_AlignedWithDirection,
  CATGSMAxisLineType_RevolutionAxis,
  CATGSMAxisLineType_MinorAxis,
  CATGSMAxisLineType_NormalToDirection,
  CATGSMAxisLineType_NormalToCircle,
  CATGSMAxisLineType_NormalToEllipse,
  CATGSMAxisLineType_NormalToOblong
}
Type for Axis Line.
Role: Used for Axis Line creation and modifications
Values:
CATGSMAxisLineType_MajorAxis
Axis created will be along major axis
CATGSMAxisLineType_AlignedWithDirection
Axis created will be aligned with direction specified
CATGSMAxisLineType_RevolutionAxis
Axis created will coincide with revolution axis
CATGSMAxisLineType_MinorAxis
Axis created will be along minor axis
CATGSMAxisLineType_NormalToDirection
Axis created will be normal to direction specified
CATGSMAxisLineType_NormalToCircle
Axis created will be along normal to plane of circle
CATGSMAxisLineType_NormalToEllipse
Axis created will be along normal to plane of ellipse
CATGSMAxisLineType_NormalToOblong
Axis created will be along normal to plane of oblong
