CATMecModUseItf Framework: CATIMmiUsePartColorsTab Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIMmiUsePartColorsTab

Interface to manage Part Colors tab values.
Use Queryinterface on MechanicalPart feature. CATPart or 3DShape needs to be saved in order to keep modifications.


Method Index
DefineGPSynchroMode(CATMmrGraphicPropertiesModes::MmrGraphicPropertiesReportMode) Modifies the way Colors are reported during CCP As Result/As Result with link process on current MechanicalPart.
GetCurrentGPSynchroMode(CATMmrGraphicPropertiesModes::MmrGraphicPropertiesReportMode&) retrieves the way Colors are reported during CCP As Result/As Result with link process on current MechanicalPart.

Methods


public virtual DefineGPSynchroMode( iSynchroMode)
Modifies the way Colors are reported during CCP As Result/As Result with link process on current MechanicalPart.
Role: this method allows user to modify the way Colors will be reported on pasted feature during CCP As Result/As Result with link process
You can set different one value as color report on part that will contain the pasted feature.
Parameters:
iSynchroMode
[in] The value to define for color report on current MechanicalPart.
Legal values: CATMmrGraphicPropertiesModes enum values.
Returns:
S_OK
The iSynchroMode has been updated successfully on the Part.
E_FAIL
The iSynchroMode has not been changed on the Part.
public virtual GetCurrentGPSynchroMode( oSynchroMode)
retrieves the way Colors are reported during CCP As Result/As Result with link process on current MechanicalPart.
Role: this method allows user to retrieve the way Colors will be reported on pasted feature during CCP As Result/As Result with link process
Parameters:
oSynchroMode
[out] The value used for color report on current MechanicalPart
Legal values: CATMmrGraphicPropertiesModes enum values.
Returns:
S_OK
The iSynchroMode has been retrieved on the Part.
E_FAIL
The iSynchroMode has not been retrieved on the Part.
