CATIAV4Interfaces Framework:CATV4IV5V4InternalCurveCreationEnum Enumeration

enum CATV4IV5V4InternalCurveCreationEnum {
  AllCurvesAreCreated,
  OnlyConicsAreCreated,
  NoInternalCurveIsCreated
}
The internal curves' creation mode during Save As Model.
Values:
AllCurvesAreCreated
All V4 curves are created in the model.
OnlyConicsAreCreated
Only internal conics are created.
NoInternalCurveIsCreated
None of the curves associated to internal face boundaries is created.
