CATIAV4Interfaces Framework:CATV4iGirwdi Global Function


CATV4iGirwdi( const iDitto, oDetail, oTrans, ier)
Analyzes a space ditto.
Parameters:
iDitto
[in] The ditto
oDetail
[out] The detail corresponding to the given iDitto
oTrans
[out] The matrix of the transformation (isometric transformation + scalling)
X1 being the coordinates of a point of the detail, in the 3-axis system of this detail ;
X2 being the coordinates of the same point, relative to the 3-axis system of the current workspace, after the ditto has been positioned.
then X2=oTrans*X1
ier
[out] error code
