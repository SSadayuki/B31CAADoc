CATIAV4Interfaces Framework:CATV4iPibrte Global Function


CATV4iPibrte( const iElement, oTypext, ier)
Retrieves the connector end type. Role: This method retrieves the end type (integer value) for the given connector element. The correspondence between the integer value and the end type name is given in the PRJ file of the CATIA V4 Model.
Parameters:
iElement
[in] The pointer to the V4 tubing connector.
oTypext
[out] The end type of the connector.
ier
[out] The error code.
Returns:
  • 0 if succeeded
  • -1 otherwise
