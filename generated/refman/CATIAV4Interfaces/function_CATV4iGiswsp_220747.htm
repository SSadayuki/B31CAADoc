CATIAV4Interfaces Framework:CATV4iGiswsp Global Function


CATV4iGiswsp( const iPrevWsp, oNextWsp, oType, oEnd, ier)
Scans the set of a workspace.
Parameters:
iPrevWsp
[in] The previous worspace
oNextWsp
[out] The following worspace.
oType
The type of workspace
Legal values:
  • 1: MASTER
  • 2: DRAW detail
  • 3: SPACE detail
  • 4: SACE detail without any DRAW elements
oEnd
[out] The end of the scan.
Legal values: value is 1 if iPrevWsp is the last element
ier
[out] error code
