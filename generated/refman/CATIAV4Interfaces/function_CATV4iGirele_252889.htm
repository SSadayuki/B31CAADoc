CATIAV4Interfaces Framework:CATV4iGirele Global Function


CATV4iGirele( const iElement, oPerm, oNum, oWsp, oSet, oPlan, ier)
Retrieves the logical information of an element.
Parameters:
iElement
[in] The V4 element
oPerm
[out] state of the element
Legal values: 1 if the element is permanent, 0 if not
oNum
[out] Number of the passive overlay model to which it belongs (0 for active model)
oWsp
[out] The element belongs to the current workspace.
Legal values: 1 for current workspace, 0 in the other case
oSet
[out] The element belongs to the current geometric set.
Legal values: 1 for current set, 0 in the other case
oPlan
[out] The element belongs to the current background plane.
Legal values: 1 for current background plane, 0 in the other case
ier
[out] error code
