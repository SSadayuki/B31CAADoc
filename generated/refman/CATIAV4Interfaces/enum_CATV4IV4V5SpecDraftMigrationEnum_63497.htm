CATIAV4Interfaces Framework:CATV4IV4V5SpecDraftMigrationEnum Enumeration

enum CATV4IV4V5SpecDraftMigrationEnum {
  squareMode,
  coneMode
}
The mode of migration of draft during Copy/Paste As Spec.
Values:
squareMode
Drafts are migrated using the square mode for ribbon progation. This mode is the mode used by default by V4 and Copy/Paste As Spec.
coneMode
Drafts are migrated using the cone mode for ribbon progation. This mode is the mode used by default by V5.
