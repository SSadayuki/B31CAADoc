AdvancedTopologicalOpe Framework:CATTopBlendCouplingMode Enumeration

enum CATTopBlendCouplingMode {
  CATTopBlendCouplingMode_DomainG0,
  CATTopBlendCouplingMode_DomainG1,
  CATTopBlendCouplingMode_DomainG2,
  CATTopBlendCouplingMode_Edge,
  CATTopBlendCouplingMode_Spine,
  CATTopBlendCouplingMode_Develop,
  CATTopBlendCouplingMode_NoTwist
}
Coupling modes.
Values:
CATTopBlendCouplingMode_DomainG0
Standard coupling: by curvilinear length.
CATTopBlendCouplingMode_DomainG1
G1 coupling: in each domain, coupling of the vertices of G0 and not G1 continuity.
CATTopBlendCouplingMode_DomainG2
G1 and then G2 coupling: after a G1 coupling, the G2 coupling is done (vertices of G1 and not G2 continuity) in each domain defined by the G1 coupling.
CATTopBlendCouplingMode_Edge
Vertex coupling: coupling is carried out edge-by-edge. You must have the same number of vertices on each wire.
CATTopBlendCouplingMode_Spine
Need to use the SetSpine Option
CATTopBlendCouplingMode_Develop
Points are coupling to minimize Gaussian Curvature of the surface Only with Blend G0G0 continuity
CATTopBlendCouplingMode_NoTwist
NOT TO BE USED.
