CATTPSUIResources Framework:CreateTPSHeadersForVisualization Global Function


CreateTPSHeadersForVisualization()
The following method allow to create all the TPS command headers that are needed for visualization of annotations. This method can be called by a workbench of Part, Product or Process workshop that want to offer annotation visualization capabilities.
