InteractiveInterfaces Framework: CATISelectionSet Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATISelectionSet

Interface to manage the elements of a Selection Set.

Role: A Selection Set is created, found and deleted with .
A Selection Set contains elements ( ). An element is a feature or a sub-element (face, edge, vertex, ...) and is refered by a CATPathElement.
