AdvancedMathematics Framework:CreateCombination Global Function


CreateCombination( const iN, const iFunc, const iCoef, const iConstant= 0.0)
Returns a function representing the linear combination of other CATMathFunctionXYs. iConstant + Sum_{i=0}^{iN-1} iCoef[i] * iFunc[i].
WARNING : the returned pointer must be deleted by the caller
This object is included in the file: CATMathFunctionXY.h
If needed, your Imakefile.mk should include the module: CATAdvancedMathematics
