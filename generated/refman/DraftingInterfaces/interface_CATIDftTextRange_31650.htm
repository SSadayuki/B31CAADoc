DraftingInterfaces Framework: CATIDftTextRange Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIDftTextRange

Manage texts by selection in this text.


Method Index
GetLength(int*) Get the number of character of the text range.
GetStart(int*) Get the start index of the text range.
GetString(wchar_t**) Get the text of the current selection.
GetTextProperties(CATIDftTextProperties**) Returns a Properties interface on the text range.
InsertSymbol(wchar_t*,int,int,wchar_t*)
InsertSymbol(CATUnicodeString&,int) Manage symbol.
SetString(wchar_t*) Modify the current text range with a new string.
