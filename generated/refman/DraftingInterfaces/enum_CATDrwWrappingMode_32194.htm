DraftingInterfaces Framework:CATDrwWrappingMode Enumeration

enum CATDrwWrappingMode {
  CATDrwWrappingOff,
  CATDrwWrappingOn,
  CATDrwWrappingAuto
}
Text wrapping modes.
Values:
CATDrwWrappingOff
Turn the text wrapping to off. The text will have no imposed width.
CATDrwWrappingOn
Turn the text wrapping to on.
CATDrwWrappingAuto
Set the text width to the current width.
