DraftingInterfaces Framework: CATIDrwGenDrawShape Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIDrwGenDrawShape

Interface to manage a drawing generated shape.
Role: A generated shape is a shape created in generative section view.


Method Index
GetBody(CATBody**) Gets the body of origin.
GetDescription(int&,CATListOfInt&,CATListPtrCATMathPoint2D&) Returns the charateristics of the generated shape.
GetPattern(CATIDftPattern**) Returns the Pattern associated to the generated shape.
GetProduct(IID&,IUnknown**) Gets the product of origin Role: Gets the product including CATPart.
