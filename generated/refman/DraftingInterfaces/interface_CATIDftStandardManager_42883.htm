DraftingInterfaces Framework: CATIDftStandardManager Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIDftStandardManager

Interface which manages the available standards for a drawing document.


Method Index
GetAvailableStandards(CATIStringList**) Description : Gets the list of the standards that can be used in the drawing document.
GetDefaultStandard(wchar_t**) Description : Gets the current standard name in the drawing document.
GetDefaultStandardPtr(CATIDftStandard**) Description : Gets the current standard object in the drawing document.
GetStandard(wchar_t*,CATIDftStandard**) Description : Gets the standard object from its name.
GetUsedStandards(CATIStringList**) Description : Gets the list of the standards that are used in the drawing document.
ImportStandard(wchar_t*) Description : Creates a standard in the drawing document and makes it current.
