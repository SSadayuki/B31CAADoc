DraftingInterfaces Framework: CATIDftFormat Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIDftFormat

Role: CATIDftFormat is an interface dedicated to get the generic informations carried by a format that are consequently shared among sheets associated with this format.


Method Index
GetFormatHeight(double&) Retrieves the height of the format.
GetFormatName(CATUnicodeString&) Retrieves the name of the format.
GetFormatOrientationsType(CATFormatOrientationsType&) Retrieves the format orientations type.
GetFormatWidth(double&) Retrieves the width of the format.
