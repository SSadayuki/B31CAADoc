enum CATDrwTextSetting {
  CATDrwBorderMode,
  CATDrwAutoFlip,
  CATDrwLineSpacing,
  CATDrwCharSlant,
  CATDrwCharSpacing,
  CATDrwLineSpacingMode,
  CATDrwMirroring,
  CATDrwFontPitch,
  CATDrwUnitDisplay,
  CATDrwLinkedScale,
  CATDrwBackField,
  CATDrwSuperScriptOffset,
  CATDrwSuperScriptSize,
  CATDrwSubScriptOffset,
  CATDrwSubScriptSize,
  CATDrwBlankingActivity,
  CATDrwDisplayMode,
  CATDrwVersion
}
Text properties.
These properties apply to whole text. Do not use not documented values.
Values:
CATDrwBorderMode
Allow to choose border with different margin.
CATDrwAutoFlip
Allow to choose if the text will flip or not when rotated beyond 90 degrees.
CATDrwLineSpacing
Allow to choose the line spacing.
CATDrwLineSpacing
Allow to choose the character slant for slantable fonts (ie. stroke fonts).
CATDrwCharSpacing
Allow to choose the inter character spacing.
CATDrwFontPitch
Allow to choose if the font is fixed or variable
