public class CATOsmExtensionServices

Services to manage former extension features and user extensions.
Role: Class of service to activate all former extension features and retrieve user extensions associated to a base feature defined in a CATfct catalog. If you have to manage existing documents containing former extension features you have to use the CATActivateExtensions services to activate your extensions. Activation is not needed for new extension features created with . User extensions are automatically created and added on a base feature when it is created. This is not a part of the interface as it is not neccessary to have the base feature itself
its type is sufficient to retrieve this informations.
