CATPDMReconcile Framework: CATIPDMUERecEventScm Interface

Usage: you can freely reimplement this interface.

interface CATIPDMUERecEventScm

This interface allows the customer to be called back.
Role: This interface allows the customer to be called back. at specific phases of the Reconciliation process (Reconciliation session initialization, etc).
You should implement it on the PDMUERecEvent component by using the CATObject mechanism.

public virtual OnAfterReconciliationApply( ixScmSession)

HRESULT Error code of function.
