Visualization Framework: CATVRDeviceEvent

Usage: you must use this class as is. You should never derive it.

public class CATVRDeviceEvent

Class that contains all the data about one event of any device.

Role: this class represents one event that is emitted by the drivers of peripherals.


Constructor and Destructor Index
~CATVRDeviceEvent() Destructor non iteratif.
Method Index
GetBooleanSize() Returns the number of booleans contained in this event.
GetBooleanValue() Returns an array containing all booleans contained in this event.
GetDeviceDriverID() Returns the ID of the device driver that had emitted this DeviceEvent.
GetDoubleSize() Returns the number of doubles contained in this event.
GetDoubleValue() Returns an array containing all doubles contained in this event.
GetEventName() Returns the name of the device event.
GetIntegerSize() Returns the number of integers contained in this event.
GetIntegerValue() Returns an array containing all integers contained in this event.
GetTimestamp() Returns the time when the event was generated.
