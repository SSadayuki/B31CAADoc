Visualization Framework: CATVREventIterator

Usage: you must use this class as is. You should never derive it.

public class CATVREventIterator

Class to directly access the device driver events in the event set structure.
Role : Whereas the gives you access to the event on every idle, the CATVREventIterator allows you to directly access the event set structure anytime. The iterator also acts as a filter so that you can only access the events whose types were specifified at the object construction. You iterate on a frozen event set. Call to update that frozen set with the newly arrived events. You may access the event set from any thread.

Update the frozen iteration event set on the dynamic event set.
At anytime, you may only access the events posted between the two last calls to Synchronize. This call is time consuming. You should not use it at a frequency high above the draw rate.
