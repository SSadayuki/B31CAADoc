Usage: you must use this class as is. You should never derive it.

public class CATLPFKRemoteControl

Class to manage the lights of a connected LPFK.


Method Index
GetLight(int) Gets the status of a given light on the LPFK.
GetLights(void) Returns the status of all lights.
SetLight(int,int) Lights and Unlights the lpfk buttons.
SetLights(unsigned int) Set a number of lights.
