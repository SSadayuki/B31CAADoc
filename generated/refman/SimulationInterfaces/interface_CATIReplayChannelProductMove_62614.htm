SimulationInterfaces Framework: CATIReplayChannelProductMove Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIReplayChannelProductMove

Edit a channel product move.

A product move channel describes the evolution of the position of a product over time. It is made of samples : each sample contains the position of the product at a given time.


Method Index
AddSample(double,double[12]) Add a sample to the channel.
GetProduct(CATBaseUnknown**) Get the product which is moved.
GetSample(int,double[12]) Read the position of one sample of the channel.
RemoveSample(int) Remove one sample of the channel.
