CATTPSInterfaces Framework: CATITPSNumericalDisplayFormat Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATITPSNumericalDisplayFormat

Interface to manage numerical format properties.

Role: This interface is available for GDT, Datum Target and Datum system.It gives the ability to retrieve or edit the numerical format properties.
