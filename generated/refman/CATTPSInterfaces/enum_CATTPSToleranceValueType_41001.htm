CATTPSInterfaces Framework:CATTPSToleranceValueType Enumeration

enum CATTPSToleranceValueType {
  CATTPSTolNumericalValue,
  CATTPSTolTabulatedValue,
  CATTPSTolNonUniformValue,
  CATTPSTolLinearNonUniformValue
}
Enum to describe tolerance value type for geometrical specifications.
Values:
CATTPSTolNumericalValue
Numerical value is used to specify tolerance zone.
CATTPSTolTabulatedValue
Tabulated value is used to specify tolerance zone.
CATTPSTolNonUniformValue
Non uniform value is employed to specify tolerance value.
In this case, tolerance zone is explicitly given by 2 surfaces one for each bound of the tolerance zone.
CATTPSTolLinearNonUniformValue
Linear non uniform value is employed to specify tolerance value.
This object is included in the file: CATTPSToleranceValueType.h
