CATTPSInterfaces Framework:CATTPSDimLimType Enumeration

enum CATTPSDimLimType {
  CATTPSDLNotDefined,
  CATTPSDLNumerical,
  CATTPSDLTabulated,
  CATTPSDLSingleLimit,
  CATTPSDLThreadValue,
  CATTPSDLTolGenValue,
  CATTPSDLInformation,
  CATTPSDLSRorSQUpperLimit
}
Describes dimension limit type.
Values:
CATTPSDLNotDefined
to be used for initialisation purpose; when returned, such a value can possibly indicate an erroneous situation.
CATTPSDLNumerical
to state Upper (Max) and lower (Min) deviation limits.
CATTPSDLTabulated
to state ISO 286-1 or ASME B4.2 tolerance class according to the applied Standard.
CATTPSDLSingleLimit
to state Unilateral limit.
CATTPSDLThreadValue
to state Thread specific tolerance class.
CATTPSDLTolGenValue
to state General tolerances.
CATTPSDLInformation
to state Auxiliary dimension or Reference dimension (no tolerances).
CATTPSDLSRorSQUpperLimit
to state Range of sizes (SR) or Standard deviation of sizes (SQ).
