CATTPSInterfaces Framework: CATITPSToleranceZone Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATITPSToleranceZone

Interface for accessing tolerance zone informations of a TPS.
Warning CATMathSetOfVectors is a protected interface (not exposed) of Mathematics framework. Subject to change.

public virtual GetToleranceValueType( oType)
Retrieve the value type of tolerance zone.
Notice: Local enumeration type "enum ValueType {Numerical,Tabulated,NonUniform};" whose definition is reminded in this comment has been substituted by global enum with same values. For existing code, just replace "ValueType" by "CATTPSToleranceValueType" to pass successfully the compilation.
Behavioral upward compatibility is guaranteed.
Parameters:
oType
value type. returns S_OK when the value type has been correctly retrieved, returns E_FAIL otherwise.
public virtual GetValue( oValue, IndexOfBound= 1 )
Retrieves tolerance zone value (in millimeters); in case of tabulated type, returned value is the resolved value.
The index optional argument is interesting for linear non uniform tolerance to manage the "lower" (IndexOfBound = 1, default) and "upper" (IndexOfBound = 2) bounds of the linear non uniform tolerance zone.
public virtual GetValue( ospTolValue, opspLinearNonUniUpperLimitTolValue= NULL )
Retrieves tolerance zone value.
Complements method for CATTPSTolTabulatedValue value; in this context, parameter type is String while in the other situations Length parameter is employed.
Second parameter is an optional argument; when specified (not null) it addresses linear non uniform tolerance, CATTPSTolLinearNonUniformValue, to manage the "upper" bound of the linear non uniform tolerance zone.
