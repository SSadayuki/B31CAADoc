CATTPSInterfaces Framework: CATITPSSetISODefaultSpecifications Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATITPSSetISODefaultSpecifications

This interface is implemented by the Annotation Set feature.
Role: Manages default operators (size and geometrical tolerances) driven by the Annotation Set. Interface meaningful for ISO Standard only (up from 14405 revision).

Enumerated Types


GDTFeatureFilteringScope
enum GDTFeatureFilteringScope {
  All,
  Form,
  Orientation,
  Location
} Enum to define the toleranced feature filtering area possible ranges.
Parameters:
All
Filtering policy applies to All form, location, orientation and run-out specifications.
Form
Filtering policy applies to form specifications; when All is active too on the set, such an activation is a way to overload All policy for Form GDTs.
Orientation
Filtering policy applies to orientation specifications; when All is active too on the set, such an activation is a way to overload All policy for Orientation GDTs.
Location
Filtering policy applies to location specifications; when All is active too on the set, such an activation is a way to overload All policy for Location GDTs.
