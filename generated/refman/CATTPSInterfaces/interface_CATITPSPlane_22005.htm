CATTPSInterfaces Framework: CATITPSPlane Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATITPSPlane

Interface implemented by Constructed Geometry (CG) type plane.

Role this interface allows to manipulate a contructed geometry type plane. To retrieve topology linked to a constructed geometry uses CATITTRSGeometry
