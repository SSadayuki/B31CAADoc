List of values to read/set linear size specification operator as per ISO 14405-1:2016 Standard.
Values:
CATTPSLSO_NotApplicable
Value reporting erroneous usage or ISO 14405-1 Standard not supported.
CATTPSLSO_None
Value reporting unmatching usage of ISO 14405-1 Standard; most of the time appears when switching from ASME to ISO.
CATTPSLSO_SA
(SA) symbol when translated into string.
The specification operator is the average of the set of values of the local size specified by the two previous modifiers (average size).
CATTPSLSO_SM
(SM) symbol when translated into string.
The specification operator is the median value of the set of values of the local size specified by the two previous modifiers (median size).
CATTPSLSO_SD
(SD) symbol when translated into string.
The specification operator is the mean of the maximum and the minimum of the set of values of the local size specified by the two previous modifiers (mid-range size).
CATTPSLSO_SX
(SX) symbol when translated into string.
The specification operator is the maximum of the set of values of the local size specified by the two previous modifiers (maximum size).
CATTPSLSO_SN
(SN) symbol when translated into string.
The specification operator is the minimum of the set of values of the local size specified by the two previous modifiers (minimum size).
CATTPSLSO_SR
(SR) symbol when translated into string.
The specification operator is the upper limit of the difference between the maximum and the minimum of the set of values of the local size specified by the two previous modifiers (range of sizes).
CATTPSLSO_SQ
(SQ) symbol when translated into string.
The specification operator is the upper limit of the standard deviation of the set of values of the local size specified by the two previous modifiers (standard deviation of sizes).
CATTPSLSO_LP
(LP) symbol when translated into string.
The specification operator is the two-point size.
If no other specification operator is defined, the specification operator is the maximum of the set of two-point size values (the maximum two-point size).
The two-point size is a local size which corresponds to the distance between two opposite points.
CATTPSLSO_LP_SA
(LP)(SA) symbol when translated into string.
The specification operator is the average of the set of two-point size values (the average two-point size).
The two-point size is a local size which corresponds to the distance between two opposite points.
CATTPSLSO_LP_SM
(LP)(SM) symbol when translated into string.
The default specification operator for linear sizes is the median value of the set of two-point size values (median size).

CATTPSLSO_LP_SD

The spherical size is a local size which corresponds to the diameter of the maximum inscribed sphere.

CATTPSLSO_GG
(GG) symbol when translated into string.
The default specification operator for linear sizes is the maximum inscribed size (i.e. the size of the associated integral feature obtained by applying the maximum inscribed criterion).
CATTPSLSO_GX
(GX) symbol when translated into string.

CATTPSLSO_GN
(GN) symbol when translated into string.

CATTPSLSO_GC
(GC) symbol when translated into string.
The specification operator is the minimax size (i.e. the size of the associated integral feature obtained by applying the minimax (Chebyshev) criterion without constraints).
CATTPSLSO_E
(E) symbol when translated into string. The specification operator is the envelope requirement.
For external (male) features of size, it corresponds to the combination of the two-point size applied for the lower limit of size and the minimum circumscribed size applied for the upper limit of size.
For internal (female) features of size, it corresponds to the combination of the two-point size applied for the upper limit of size and the maximum inscribed size applied for the lower limit of size.
CATTPSLSO_CC
(CC) symbol when translated into string.
The specification operator is the circumference diameter.
If no other specification operator is defined, the specification operator is the maximum of the set of circumference diameters (the maximum circumference diameter).
The circumference diameter is a calculated size which corresponds to the diameter d, obtained from the formula d=C/PI, where C is the length of the integral extracted line in a cross section normal to the axis of the least-squares associated cylinder.

CATTPSLSO_CV
(CV) symbol when translated into string.
The specification operator is the volume diameter (i.e. the calculated size corresponding to the diameter d, obtained from the formula d=SQRT(4*V/(PI*L)), where V is the volume limited by the integral extracted cylinder, L is the height of the cylinder taken between two parallel planes perpendicular to the axis of the least-squares associated cylinder with the maximum distance between them and containing a complete section of the feature, and SQRT is the square root function).
CATTPSLSO_Length
The specification applies to any restricted portion of the considered feature of size.
CATTPSLSO_ACS
The specification applies to any cross section of the considered feature of size.
CATTPSLSO_ALS
The specification applies to any longitudinal section of the considered feature of size.
CATTPSLSO_SCS
The specification applies to one or several specific fixed cross sections of the complete feature of size.
Both the SCS modifier and the cross section identifiers are displayed in the 3D dimension (example: Phi 10 H7 SCS [A, B]).
CATTPSLSO_SCSndId
The specification applies to one or several specific fixed cross sections of the complete feature of size.
Only the SCS modifier is displayed the 3D dimension. The cross section identifiers are not displayed in the 3D dimension (example: Phi 10 H7 SCS).
CATTPSLSO_SCSnd
The specification applies to one or several specific fixed cross sections of the complete feature of size.
The SCS modifier and the cross section identifiers are not displayed in the 3D dimension (example: Phi 10 H7).
