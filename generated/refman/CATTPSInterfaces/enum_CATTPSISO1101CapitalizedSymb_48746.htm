List of values to read/set specification elements for form association as per ISO 1101 Standard.
Values:
CATTPS1101NotApplicable
Value reporting erroneous usage or ISO 1101 Standard not supported.
CATTPS1101None
Value reporting unmatching usage of ISO 1101 Standard; most of the time appears when switching from ASME to ISO.
CATTPS1101_P
P symbol when translated into string.
Characteristic parameter: Reference to peak.
CATTPS1101_V
V symbol when translated into string.
Characteristic parameter: Reference to valley.
CATTPS1101_T
T symbol when translated into string.
Characteristic parameter: Peak to valley.
CATTPS1101_Q
Q symbol when translated into string.
Characteristic parameter: Root mean square (RMS).
CATTPS1101ATFS_C
(C) symbol when translated into string.
Toleranced feature association symbol: Minimax feature.
CATTPS1101ATFS_G
(G) symbol when translated into string.
Toleranced feature association symbol: Least squares feature.
CATTPS1101ATFS_N
(N) symbol when translated into string.
Toleranced feature association symbol: Minimum circumscribed feature.
CATTPS1101ATFS_T
(T) symbol when translated into string.
Toleranced feature association symbol: Tangent feature.
CATTPS1101ATFS_X
(X) symbol when translated into string.
Toleranced feature association symbol: Maximum inscribed feature.
CATTPS1101_C
C symbol when translated into string.
The reference feature association criterion for form specifications is the minimax (Chebyshev) without constraints; it can be set as default on the Annotation Set.
The default parameter that applies is the total range of deviations (i.e. the distance from the lowest valley of the toleranced feature to the reference feature, plus the distance from the highest peak of the toleranced feature to the reference feature).
CATTPS1101_C_T
CT symbol when translated into string.
The reference feature default association criterion for form specifications is the minimax (Chebyshev) without constraints; Only available for local definition onto geometrical element.
Formal parameter that applies is Peak to valley.
