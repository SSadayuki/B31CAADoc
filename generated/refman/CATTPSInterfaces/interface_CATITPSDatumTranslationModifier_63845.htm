CATTPSInterfaces Framework: CATITPSDatumTranslationModifier Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATITPSDatumTranslationModifier

This interface is implemented by Datum Reference Frame features.
Role: Interface for managing Datum Feature Simulator Translation Modifier applied onto Datum Feature employed within a Reference Frame.
This interface is only valid for ASME 2009 standard (does not exist in ISO).

Important notice: this modifier is only valid for Secondary or Tertiary boxes of the DRF.

to know how to state what box in Datum Reference Frame end user is interested in.
