CATTPSInterfaces Framework: CATITPSISOTolerancedFeatureFilter Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATITPSISOTolerancedFeatureFilter

This interface is implemented by CATTPSToleranceWithDRF and CATTPSToleranceWithoutDRF features.
Role: Manages toleranced feature applied filter either localy assigned or inherited form the Annotation Set default definition. Interface meaningful for ISO Standard only (up from 1101:2017 revision).


Method Index

Role: Retrieves the applied toleranced feature filtering (either form local definition of from Annotation Set default).
SetGDTFeatureFiltering(CATUnicodeString*)
Role: Sets the toleranced feature filtering; definition is therefore becoming local (overloading) subsequently to this call.
