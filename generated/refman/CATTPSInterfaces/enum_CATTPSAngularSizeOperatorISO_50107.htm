List of values to read/set angular size specification operator as per ISO 14405-3:2016 Standard.
Values:
CATTPSASO_NotApplicable
Value reporting erroneous usage or ISO 14405-3 Standard not supported.
CATTPSASO_None
Value reporting unmatching usage of ISO 14405-3 Standard; most of the time appears when switching from ASME to ISO.
CATTPSASO_SA
(SA) symbol when translated into string.
The specification operator is the average of the set of values of the local size specified by the two previous modifiers (average size).
CATTPSASO_SM
(SM) symbol when translated into string.
The specification operator is the median value of the set of values of the local size specified by the two previous modifiers (median size).
CATTPSASO_SD
(SD) symbol when translated into string.
The specification operator is the mean of the maximum and the minimum of the set of values of the local size specified by the two previous modifiers (mid-range size).
CATTPSASO_SX
(SX) symbol when translated into string.
The specification operator is the maximum of the set of values of the local size specified by the two previous modifiers (maximum size).
CATTPSASO_SN
(SN) symbol when translated into string.
The specification operator is the minimum of the set of values of the local size specified by the two previous modifiers (minimum size).
CATTPSASO_SR
(SR) symbol when translated into string.
The specification operator is the upper limit of the difference between the maximum and the minimum of the set of values of the local size specified by the two previous modifiers (range of sizes).
CATTPSASO_SQ
(SQ) symbol when translated into string.
The specification operator is the upper limit of the standard deviation of the set of values of the local size specified by the two previous modifiers (standard deviation of sizes).
CATTPSASO_GG
(GG) symbol when translated into string.
The specification operator is the least-squares global angular size.
If the angular feature of size is a cone, it's the angular dimension of the associated integral feature (a cone) obtained by applying the least-squares (Gaussian) criterion without constraints.
If the angular feature of size is a pair of planes (a wedge) or a pair of coplanar straight lines, it's the angular dimension between the associated integral features (a pair of planes or a pair of coplanar straight lines) obtained by applying the least-squares (Gaussian) criterion without constraints (each element being associated independently)..
CATTPSASO_GC
(GC) symbol when translated into string.
The specification operator is the minimax global angular size.
If the angular feature of size is a cone, it's the angular dimension of the associated integral feature (a cone) obtained by applying the minimax (Chebyshev) criterion without constraints.
If the angular feature of size is a pair of planes (a wedge) or a pair of coplanar straight lines, it's the angular dimension between the associated integral features (a pair of planes or a pair of coplanar straight lines) obtained by applying the minimax (Chebyshev) criterion without constraints (each element being associated independently).
CATTPSASO_LC
(LC) symbol when translated into string.
The specification operator is the two-line angular size.
If no other specification operator is defined, the specification operator is the maximum of the set of two-line angular size values (the maximum two-line angular size).
The two-line angular size is a local size.
Two sequential association operations are used to define it.
a) To associate the real angular features of size with a feature of the same nature.
b) To associate the extracted lines (coming from intersection of the real angular features of size and the intersection plane) with two straight lines.
For the first association (a), we use the least squares association criterion (without constraints) for the two extracted surfaces (independently of each other) in case of a prismatic feature of size (wedge), or for the extracted surface in case of a revolute feature of size (cone).
The first association with a least squares criterion is to establish either a stable intersection line for the prismatic feature of size, or a stable axis for the revolute feature of size.
For the second association (b), we use the minimax association criterion. The two lines are associated independently of each other.

CATTPSASO_Length
The specification applies to any restricted portion of the considered feature of size.
CATTPSASO_Angle
The specification applies to any restricted portion of the considered feature of size.
CATTPSASO_SCS
The specification applies to one or several specific fixed cross sections of the complete feature of size.
Both the SCS modifier and the cross section identifiers are displayed in the 3D dimension.
CATTPSASO_SCSndId
The specification applies to one or several specific fixed cross sections of the complete feature of size.
Only the SCS modifier is displayed the 3D dimension. The cross section identifiers are not displayed in the 3D dimension.
CATTPSASO_SCSnd
The specification applies to one or several specific fixed cross sections of the complete feature of size.
The SCS modifier and the cross section identifiers are not displayed in the 3D dimension.
