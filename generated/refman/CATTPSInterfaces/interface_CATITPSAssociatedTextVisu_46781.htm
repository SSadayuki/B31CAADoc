CATTPSInterfaces Framework: CATITPSAssociatedTextVisu Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATITPSAssociatedTextVisu

Interface dedicated to manage the association of a visualization text feature with another TPS.
This interface manages the associativity between an annotation(A semantic datum or a semantic dimension) and a visualization text.


Method Index
GetTextVisu(CATITPSText**,CATUnicodeString*) Retrieves the Visualization Text associated to a TPS.
GetVisuTextTargetAndPattern(CATITPSText**) Retrieves the Visualization Text associated to a TPS for Datum Target adn Pattern.
SetTextVisu(CATUnicodeString,CATITPSText*) Sets the Visualization Text associated to a TPS.
SetVisuTextTargetAndPattern(CATITPSText*) Sets the Visualization Text associated to a TPS for Datum Target and Pattern.
