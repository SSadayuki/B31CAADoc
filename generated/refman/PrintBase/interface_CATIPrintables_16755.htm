PrintBase Framework: CATIPrintables Interface

Usage: you can freely reimplement this interface.

interface CATIPrintables

Interface for printable objects.
This interface is used for the multidocuments printing. The implementation of this interface allows to use the print dialog window for multidocuments.


Method Index
CleanPrintableImage(void) Clean the printable object environment.
CreatePrintableImages(int,int) Creates a CATPrintImage from the printable object.
GetCurrentImageNumber(int&) Gets the current image number.
GetImageName(int,CATUnicodeString&) Gets the print image name.
GetImagesNumber(int&) Gets the count of images in the document.
