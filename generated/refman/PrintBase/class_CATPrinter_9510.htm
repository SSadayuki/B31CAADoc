PrintBase Framework: CATPrinter

Usage: you must use this class as is. You should never derive it.

public class CATPrinter

Class to describe a printer and its attributes.

Role: The printer is defined by the user description, its language and its default and largest format sizes. A query on allows to obtain the list of system printers.
