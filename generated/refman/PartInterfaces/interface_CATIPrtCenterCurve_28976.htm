PartInterfaces Framework: CATIPrtCenterCurve Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIPrtCenterCurve

Interface which provides a center curve management behavior to a feature center curve.
Role: Some features contain a center curve component which allows users to define and use a center curve. The center curve internal feature is managed through this interface.


Method Index
GetBody(CATBody_var&) Gets the topological body corresponding to the center curve.
GetElement(CATISpecObject_var&) Gets the specificaton element of the center curve.
GetGeomSupport(CATGeometry_var&) Gets the geometrical support of the center curve.
SetElement(CATISpecObject_var&) Set a new element to the specifications of the center curve.
