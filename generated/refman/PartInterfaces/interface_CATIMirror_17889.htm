PartInterfaces Framework: CATIMirror Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIMirror

Interface to read or modify the mirror's specifications.
Role: The mirror feature allows users to duplicate an original body according to a symmetry.


Method Index
GetIfDesignIntent() Returns the Design Intent attribute value.
GetPlane() Returns the mirroring element as a plane or face that specifies the mirror.
ModifyMirrorPlane(CATISpecObject_var&) Sets the face or plane about which the body is mirrored.
SetDesignIntent(int) Sets the Design Intent attribute value.
