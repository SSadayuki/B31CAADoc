enum CATPrtSplitType {
  PositiveSide,
  NegativeSide
}
Side to be kept for splitting or sewing operation.
Values:
PositiveSide
It refers to the same orientation as the splitting or sewing surfacic feature normal vector.
NegativeSide
It refers to the opposite orientation as the splitting or sewing surfacic feature normal vector.
