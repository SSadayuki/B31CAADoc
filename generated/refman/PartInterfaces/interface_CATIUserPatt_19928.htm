PartInterfaces Framework: CATIUserPatt Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIUserPatt

Interface to read or modify the specifications of the user pattern.
Role: The user pattern feature allows users to duplicate an original feature according an user distribution.


Method Index
GetLocationElt() Returns the list of location elements.
GetLocationOrientations(CATLISTP(CATMathTransformation)&) Returns the list of location orientations.
GetNbCopy() Returns the number of times that the original feature will be duplicated.
ModifyLocationElt(CATISpecObject_var&) Sets the location elements.
ModifyLocationElts(CATLISTV(CATISpecObject_var)&) Sets the location elements.

Methods


public virtual GetLocationElt()
Returns the list of location elements. The list is composed of one sketch of points.
public virtual GetLocationOrientations( oListOrientTransfos)
Returns the list of location orientations. The list is composed of transofrmation containing only orientations at each location point. The list items must be deleted after use.
public virtual GetNbCopy()
Returns the number of times that the original feature will be duplicated.
public virtual ModifyLocationElt( const ihObjectList)
Sets the location elements.
Parameters:
ihObjectList
One sketch of points.
public virtual ModifyLocationElts( const iListOfLocationElts)
Sets the location elements.
Parameters:
iListOfLocationElts
List of Positioning elements, it can be either a single sketch of points or list of points/axis systems.
