Usage: you can derive this class.

public class CATDlgStatusBar

Base class for document status bar.
Role: The status bar is used to display temporary or permanent information related to the application status, such as the prompts of the active command or its messages, and a keyboard entry field that is used by the application PowerInput. It is generally located at the bottom of the window. It can also include push buttons, check buttons, radio buttons, and labels.
Use the status bar when you want to give application status information.
