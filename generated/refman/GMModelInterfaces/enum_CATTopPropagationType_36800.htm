GMModelInterfaces Framework:CATTopPropagationType Enumeration

enum CATTopPropagationType {
  CatPropagSingle,
  CatPropagExtrapolated,
  CatPropagClosing,
  CatPropagAllowPartial
}
Specifies the management of the propagation algorithm .
Values:
CatPropagSingle
No propagation is performed.
CatPropagExtrapolated
The specified geometry is extrapolated.
CatPropagClosing
Adjacent topological cells are agregated within a skin.
CatPropagAllowPartial
Allow limit to partially block the extrusion and select the first volume of intersection between the extruded body and the limiting body.
