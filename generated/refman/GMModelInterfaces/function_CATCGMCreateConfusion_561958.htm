GMModelInterfaces Framework:CATCGMCreateConfusion Global Function


CATCGMCreateConfusion( iFactory, iConfig, const iPoint1, const iPoint2, const iTol, iMode= BASIC)
Creates an operator for testing the overlap of two CATPointOnSurfaces, with respect to the geodesical length between the points.
The two points belong to the same CATSurface.
Parameters:
iFactory
The pointer to the factory of the points.
iPoint1
The pointer to the first point.
iPoint2
The pointer to the second point.
iTol
The distance for deciding the overlap.
iMode
BASIC
the operation is performed at the operator creation
ADVANCED
otherwise (not implemented yet)
Returns:
[out, IUnknown#Release] The pointer to the created operator. To be released with the Release method after use.
