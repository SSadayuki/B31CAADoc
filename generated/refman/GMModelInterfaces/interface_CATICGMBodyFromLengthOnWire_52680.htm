GMModelInterfaces Framework: CATICGMBodyFromLengthOnWire Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATICGMBodyFromLengthOnWire

Class defining an operator to create a point or a series of points at given measures (arc lengths spacings or integrated turning angles) on a Wire.


The CATBodyFromLengthOnWire operator follows the global frame of the topological operators and satisfies the smart mechanism: the input bodies are not modified. A new resulting body is created, possibly sharing data with the input bodies. The follwing two integral measures are curently supported:

  • Arc length spacing specifies the curve length between sample points.
  • Integrated turning spacing specifies the total amount the tangent vector changes between sample points. Jumps in the tangent vector between edges are ignored. The expectation is that edge pairs are G1 continuous at vertices.
  • The result is accessed with the GetResult method. If you do not want to keep the resulting body, use the method to remove it from the geometric factory.

public virtual GetLayDown( oLaid)
Returns the lay-down setting.
Parameters:
oLaid
0 if lay-down is turned off, 1 if it is turned on.
public virtual GetResult()
Returns the point body computed by the run. Call this method after the Run method has been called.
public virtual GetResult( oLocations)
public virtual GetResultWithTangents( oResultVertices, oTangents, oLocations= NULL)
Returns the result body with tangents (else returns NULL) associated to its vertices, according to the wire orientation. SetTangentComputation(TRUE) then Run() has to be called prior to the use of this method
Returns:
The oResultVertices list of sampled vertices in the Result CATBody. The oTangents list of normalized tangents associated to the vertices. The oLocations optional list of vertices locations in 3D. The pointer to the resulting body. Use if you do not want to keep it.
public virtual Run()
Runs the operator.
Returns:

Legal values: 0 if ok, 1 if failed
public virtual SetLayDown( iWireBody, ilaid)
Activates or deactivates the lay-down operation. If activated, the operator projects the resulting point body upon the specified wire body if the distance between them is lower than the factory resolution.
Parameters:
iWireBody
The wire body that is the target of the lay-down operation.
ilaid
0 to turn off lay-down, 1 to turn it on.
public virtual SetTangentComputation( iActivateTangentMode)
Activates or deactivates the computation of tangents at the result vertices. If activated, the operator evaluates the tangent on the input wire, or at its extremity in case of extrapolation.
Parameters:
iActivateTangentMode
FALSE to turn off the option, TRUE to turn it on.
