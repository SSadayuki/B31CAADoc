CATGbfModelInterfaces Framework: CATIABFExport Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIABFExport

Interface to Export fastening data in a neutral format file (.
txt file).
Role: Export ABF data under a reference product to ABF neutral file.

  • QueryInterface on product reference to create a pointer to this interface.
  • Release this interface when it is no longer needed.

Methods


public virtual Export()
Method generates neutral file(.txt) for the product on which it is called. Also it generates log file (.xml) containing warning(s) about the fasteners for which incomplete data is exported. at the same location as neutral file.
Returns:
S_OK
if export operation is successful.
E_INVALIDARG
If any input set by user for export operation is Invalid.
E_FAIL
if file is already existing at the path or not writable file or failure of Export due to any other reason.
public virtual SetAdditionalInfoFlag( const ibIncludeAdditionalInfo)
Sets Flag for additional information.
Parameters:
ibIncludeAdditionalInfo
[in] Flag to indicate if additional information (such as fastener projections) should be exported (TRUE or FALSE). Default value = TRUE.
Returns:
S_OK
public virtual SetCommentsFlag( const ibIncludeComments)
Sets Flag for Comments.
Parameters:
ibIncludeComments
[in] Flag to indicate if commented header block should begin the export file (TRUE or FALSE). Default value = TRUE.
Returns:
S_OK
public virtual SetDiscretizationMethod( const iDiscMethod)
Sets the discretization method to represent curvilinear fasteners.
Parameters:
iDiscMethod
[in] The discretization method. Two possible values: "sag", "step". Default value = "sag".
Returns:
S_OK
if valid discretisation method.
E_INVALIDARG
otherwise
public virtual SetDiscretizationTolerance( const iTolerance)
Sets the sag for polyline. The input string is not case sensitive.
Parameters:
iSagvalue
[in] Sag value for Polyline. Default value = 0.2.
Returns:
S_OK
if valid input. Valid range: 0 < Sag <= 10.
E_INVALIDARG
if the Sag value is not in valid range.
public virtual SetOutputFilePath( const iExportDirectoryPath, const iExportFileName)
Sets the full Directory path and the File name of the output export file.
Parameters:
iExportDirectoryPath
[in] The Directory path where file is to be exported. Default Value = System temp Directory.
iExportFileName
[in] The Export neutral file name. Default Value = reference product name(e.g. Product.txt).
Returns:
S_OK
if the Directory path and File name are valid.
E_INVALIDARG
if File name or Directory path is empty.
E_FAIL
if File name already exists at given Directory path or Directory is inaccessible or File isn't writable.
public virtual SetPrecision( const iNbDigits)
Sets the precision used to format real numbers.
Parameters:
iNbDigits
[in] The number of significant digits. Default value = 6.
Returns:
S_OK
if valid input. Valid range: 0 <= iNbDigits <= 10.
E_INVALIDARG
if iNbDigits is not in range.
public virtual SetScientificFlag( const ibScientificFlag)
Sets the Flag for Scientific notations.
Parameters:
ibScientificFlag
[in] Flag to indicate whether to use the Scientific notation (TRUE or FALSE). Default value = FALSE.
Returns:
S_OK
public virtual SetUnit( const iUnit)
Sets the unit of the export file. The input string is not case sensitive.
Parameters:
iUnit
[in] unit name - Two possible values: "mm" and "inch". (mm for millimeter). Default value = "mm".
Returns:
S_OK
if valid unit name.
E_INVALIDARG
otherwise.
