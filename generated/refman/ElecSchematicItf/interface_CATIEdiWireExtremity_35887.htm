ElecSchematicItf Framework: CATIEdiWireExtremity Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIEdiWireExtremity

Interface to manage an electrical schematic Wire Extremity.
Role:This interfaces describes the electrical object Wire Extremity.


Method Index
Connect(CATIEdiPin*) Connects the wire extremity to a Pin Connector.
Disconnect() Disconnects the cable extremity.
GetCableExtremity(CATIEdiCableExtremity**) Gets the calbe extremity in accordance with this wire extremity.
GetConnectedPin(CATIEdiPin**) Gets Connected Pin connector.
GetNumber(int*) Get the number of the wire extremity.
GetWire(CATIEdiWire**) Gets Wire electrical object.
