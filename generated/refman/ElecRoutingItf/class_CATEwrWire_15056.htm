ElecRoutingItf Framework: CATEwrWire

Usage: you must use this class as is. You should never derive it.

public class CATEwrWire

This class is used to describe a wire.


Data Member Index
BendRadius The BendRadius attribute corresponds to the maximum curvature of the wire.
Color The color of the wire.
FatherWireGroup The identifier of the WireGroup (if exists) immediate father of the wire.
Id The identifier of the wire.
Id_FirstExtremity The identifier of the first extremity.
Id_Harness The identifier of the harness (if exists) containing the wire.
Id_SecondExtremity The identifier of the second extremity.
InnerDiameter The InnerDiameter attribute corresponds to the diameter of the conductor, excluding the insulation.
LinearMass The LinearMass attribute corresponds to the linear mass of the wire.
Name The name of the wire.
OuterDiameter The OuterDiameter attribute corresponds to the diameter of the wire, including the insulation.
PartNumber The PartNumber attribute corresponds to the part number of the wire.
SeparationCode The SeparationCode attribute is used in knowledge rules checked in wire routing.
SubType The subtype of the wire.
sSignalId The SignalID of the wire routed.
