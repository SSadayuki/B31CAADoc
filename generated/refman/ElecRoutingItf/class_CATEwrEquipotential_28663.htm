ElecRoutingItf Framework: CATEwrEquipotential

Usage: you must use this class as is. You should never derive it.

public class CATEwrEquipotential

This class is used to describe an equipotential.


Data Member Index
Diameter The Diameter attribute corresponds to an estimation of the diameter of the wires which will realize the equipotential.
Extremities This list contains the identifier of the extremities of the equipotential.
Id The identifier of the equipotential.
Name The name of the equipotential.
RoutingPriority The RoutingPriority attribute is used in equipotential routing.
SeparationCode The SeparationCode attribute is used in knowledge rules checked in equipotential routing.
SubType The subtype of the equipotential.
WirePartNumber The WirePartNumber attribute corresponds to the part number of the wire which will realize the equipotential.
