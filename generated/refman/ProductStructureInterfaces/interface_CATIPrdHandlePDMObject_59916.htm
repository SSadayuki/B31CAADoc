interface CATIPrdHandlePDMObject

Interface to manage the impacts of modifcation in PDM context.

Role : By using this interface for setting properties on product, impacted elements are logged in PDM context.

Warnings :

For non-PDM scenario user can use CATIProperty interface.

This interface is implemented on ASMProduct features only.

Implementation is only avalaible under PX1 license.
