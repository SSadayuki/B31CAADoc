interface CATICfgSpecExpression

Interface for Specification Expression.

GetUIDList(SEQUENCE(CORBAAny)&) Gets the list of Configured Objects on which a Spec Expression is attached to.
GetValue(CATICfgORExpression_var&) Gets the value of a Specification Expression.
SetDescription(CATUnicodeString&) Sets the description of a Specification Expression.
SetValue(CATICfgORExpression_var&) Sets the value of a Specification Expression.
Solve(CATListValCATICfgSimpleSpecification_var&) Checks whether the value is true or not depending on a list of Simple Specifications.
