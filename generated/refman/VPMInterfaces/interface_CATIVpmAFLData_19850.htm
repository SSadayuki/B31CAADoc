VPMInterfaces Framework: CATIVpmAFLData Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIVpmAFLData

Interface for AFLData objects.
Role: The interface provides functions for accessing AFLData attributes.


Method Index
Dump(char*) Dump the Data attributes.
GetCreationDate(CATTime&) Get the creation date.
GetDataValue(CATUnicodeString&) Get the value of the Data.
GetDescription(CATUnicodeString&) Get the description of the Data.
GetIdentifier(SEQUENCE(octet)&) Get the binary reference to the actual Data object.
GetLastModifDate(CATTime&) Get the last modification date.
GetName(CATUnicodeString&) Get the name of the Data.
SetDataValue(CATUnicodeString&) Set a value as a string to the Data.
SetDescription(CATUnicodeString&) Set the description of the Data.
SetIdentifier(SEQUENCE(octet)&) Set the binary reference to the actual Data object.
SetName(CATUnicodeString&) Set the name of the Data.
