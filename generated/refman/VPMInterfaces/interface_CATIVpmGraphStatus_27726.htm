VPMInterfaces Framework: CATIVpmGraphStatus Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIVpmGraphStatus

This interface is implemented by the status objects of the LCA lifecycle status graph model.
