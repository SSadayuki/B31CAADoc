VPMInterfaces Framework: CATIVpmAFLLink Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIVpmAFLLink

Interface for AFLLink objects.
Role: The interface provides functions for accessing AFLLink attributes.


Method Index
Dump(char*) Dump the Link attributes.
GetCreationDate(CATTime&) Get the creation date.
GetFromAction(CATIVpmAFLAction_var&) Get the source Action of this Link.
GetLastModifDate(CATTime&) Get the last modification date.
GetLevel(long) Get the level of the Link.
GetName(CATUnicodeString&) Get the name of the Link.
GetToAction(CATIVpmAFLAction_var&) Get the target Action of this link.
GetType(CATUnicodeString&) Get the Link type.
GetValue(CATUnicodeString&,CORBAAny&) Get any attribute of a Link.
SetLevel(long) Set the level of the Link.
SetName(CATUnicodeString&) Set the name of the Link.
SetValue(CATUnicodeString&,CORBAAny&) Set any attribute of a Link.
