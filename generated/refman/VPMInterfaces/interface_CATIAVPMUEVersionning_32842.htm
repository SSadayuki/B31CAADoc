interface CATIAVPMUEVersionning

Represents the Versionning possibilities User Exit.
There is a User Exit for each case : - GetNewVersion - GetPreferedVersion


Method Index

GetPreferedVersion(CATIAVPMObjectVersion*&) User Exit called when an ObjectVersion is queried from a CATIAVPMObjectMaster.
