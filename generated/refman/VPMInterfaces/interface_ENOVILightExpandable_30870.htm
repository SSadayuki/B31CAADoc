public virtual getLightTree( iExpandLevel, iSelectClause, oResult, iType, iDocCriteria= LEX_NoDocuments, iTechPackCriteria= FALSE, const iCfgFilter= NULL_var )
Expands a root object, returning only light structures (no object is loaded in session).
Parameters:
iExpandLevel:
level of expand to go to. - LEX_MODE_ALL: expand all and continue to expand Product Instances. - LEX_MODE_42: expand all but do not expand Product Instances further. - LEX_MODE_DMU: expand all returning only the matching objects to the filter. - 1, 2, 3...: expand to the specified level.
iSelectClause:
Additional attributes to get the values from.
oResult:
result of the expand, an ordered list of light structures.
iType:
type of the tree to get (LEX_ItemInstance, LEX_GenericComponent or LEX_ProductComponent).
iDocCriteria:
type of the documents to get on the parts.
iTechPackCriteria:
retrieves or not Technological Packages.
iCfgFilter:
Configuration filter.
Returns:
S_OK if everything went fine E_FAIL if there was an error
