Macro OBJECT_DATA_EVENT_INIT (iObject,EVENT_TYPE,oStream)
Macro to get a pointer on a publisher data stream to be filled according to XML definition.
To be used in cases where the event method code is put outside the firing object implementation. Implies that an appropriate object instance is valid in the scope of that code.
Parameters:
iObject
the object that will publish the event
EVENT_TYPE
the Event name which the stream is associated with: must be written as a symbol, not a string
oStream
a smart pointer on the empty stream to be filled before the event is fired. NB : When the event is fired, the system associates automatically the filled stream with this event.
This object is included in the file: ENOVDataEventMacro.h
