NewTopologicalObjects Framework:CATCheckTopology Global Function


CATCheckTopology( iObject, ioNumberOfSevereAnomalies, ioForDumpOfAnomalies)
Performs the topological check of a CGM object.
Parameters:
iObject
The pointer to the object to check.
ioNumberOfSevereAnomalies
The count of detected anomalies.
ioForDumpOfAnomalies
The stream on which the anomalies are dumped.
Returns:
The stream on which the anomalies are dumped (same as ioForDumpOfAnomalies).
