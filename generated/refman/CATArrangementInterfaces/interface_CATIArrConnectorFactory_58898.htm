CATArrangementInterfaces Framework: CATIArrConnectorFactory Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIArrConnectorFactory

Interface to manage connectors on untyped parts as in the case for SRT, PLO, or SSR Role: To manage connectors on spatial objects.
