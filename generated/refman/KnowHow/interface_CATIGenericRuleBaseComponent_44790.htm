KnowHow Framework: CATIGenericRuleBaseComponent Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIGenericRuleBaseComponent

Interface dedicated to defining the basic properties of the Knowledge Expert hierarchy of rulebase, rulesets, checks and rules.


Method Index
CheckInContext() Parses and adds the component to the inference engine.
GetActivate() Gets the activity of the component.
GetFatherRuleSet() Gets the ruleset the component belongs directly to.
GetGenericRuleBaseComponentName() Gets the name of the component.
GetInferenceEngineType() Gets the type of the inference engine.
GetRuntimeBase() Gets the rulebase the component belongs (directly or through rulesets) to.
IsUseOnly() Gets the "use-only-ness" of the component.
SetActivate(CATCke::Boolean) Sets the activity of the component.
SetGenericRuleBaseComponentName(CATUnicodeString) Sets the name of the component.
SetInferenceEngineType(CATExpertRuleEnum::InferenceEngineType) Sets the type of the inference engine.
SetUseOnly() Forces the component to be use-only (not editable, not removable).
