CATAnalysisInterfaces Framework:SAMHiddenState Enumeration

enum SAMHiddenState {
  SAMHiddenStateTrue,
  SAMHiddenStateFalse,
  SAMHiddenStateUnknown
}
Visibility state of a Basic Component frame.
Role:SAMHiddenState defines the visibility of a basic component frame.
Values:
SAMHiddenStateTrue
The frame is hidden.
SAMHiddenStateFalse
The frame is shown.
SAMHiddenStateUnknown
No impact on the frame. The frame will decide by itself. (Default Value)
