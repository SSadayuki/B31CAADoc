CATAnalysisInterfaces Framework: CATSamDocNotification

Usage: you must use this class as is. You should never derive it.

public class CATSamDocNotification

Class representing a notification send when the analysis document is loaded of saved.
Role: An instance of this class is sent by the CATAnalysis document when the load or save operation are performed.
