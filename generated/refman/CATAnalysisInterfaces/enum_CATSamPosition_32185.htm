CATAnalysisInterfaces Framework:CATSamPosition Enumeration

enum CATSamPosition {
  CATSamPositionNode,
  CATSamPositionElement,
  CATSamPositionCenter,
  CATSamPositionDof,
  CATSamPositionNodeOfElement,
  CATSamPositionEdge,
  CATSamPositionFace,
  CATSamPositionGauss,
  CATSamPositionVirtualNode,
  CATSamPositionVirtualElement,
  CATSamPositionUserDefined,
  CATSamPositionGroup,
  CATSamPositionSet,
  CATSamPositionLast
}
The positions on which field data entities can be applied to.
Values:
CATSamPositionNode
A node.
CATSamPositionElement
An element.
CATSamPositionCenter
The center of an element.
CATSamPositionDof
A degree of freedom.
Note : all positions defined after this one (and none before) need a number to be specified with the apply definition
. The valid numbers for this position are defined by CATSamDof.
CATSamPositionNodeOfElement
A node of an element.
CATSamPositionEdge
An edge of an element.
CATSamPositionFace
A face of an element.
CATSamPositionGauss
A gauss point of an element.
CATSamPositionVirtualNode
A virtual node.
CATSamPositionVirtualElement
A virtual element.
CATSamPositionUserDefined
An user defined position.
CATSamPositionGroup
A Group position.
CATSamPositionSet
A Set position.
CATSamPositionLast
No position type defined.
It also represents the number of valid enumerated values.
