CATAnalysisInterfaces Framework:CATSamVisuMode Typedef

typedef int CATSamVisuMode
Enumerate to manage visualization modes in the window definition.
Values can be cumulated to multiple type authorized.
Values:
CATSamVisuModeMeshing:
Mesh is visualized.
CATSamVisuModeGeometry:
Geometry is visualized.
CATSamVisuModePreProcessing:
Preprocessing is visualized.
CATSamVisuModePostProcessing:
Images are visualized.
CATSamVisuModeRealDeformation:
Real deformation image is visualized.
This object is included in the file: CATSamAnalysisContext.h
