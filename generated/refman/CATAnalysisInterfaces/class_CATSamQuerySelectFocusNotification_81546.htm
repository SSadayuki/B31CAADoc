CATAnalysisInterfaces Framework: CATSamQuerySelectFocusNotification

Usage: you must use this class as is. You should never derive it.

public class CATSamQuerySelectFocusNotification

Base class for notifications related to Dialog agent acquisition.
Role: This modification is linked with the implementation. It is dedicated to warn the Edition Command of the Analysis Entity when it must delegate the selection management to the implementing command and when this delegation is ended.
