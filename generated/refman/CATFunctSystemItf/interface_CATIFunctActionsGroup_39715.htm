interface CATIFunctActionsGroup

The interface to access a group of functional actions in a system.


Method Index
Add(CATIFunctAction*) Adds an action to the actions' group.
GetActions() Get the list of Functional Actions included in the group.
GetExtremities(double&,double&,double&,double&) Get coordinates of Input and Output extremities.
RemovePosition(long) Removes an action from the actions' group.
SetExtremities(double,double,double,double) Set coordinates of Input and Output extremities.
