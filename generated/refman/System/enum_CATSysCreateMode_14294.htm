System Framework:CATSysCreateMode Enumeration

enum CATSysCreateMode {
  CATSys_OpenExisting,
  CATSys_CreateNew,
  CATSys_CreateAlways
}
The possible actions to take on documents that exist and which action to take when documents do not exist.
Values:
CATSys_OpenExisting
opens the document, but fails if it does not exist already.
CATSys_CreateNew
creates a new document but fails if it already exists.
CATSys_CreateAlways
creates a new document even if it already exists. In this case it will be overwriten.
