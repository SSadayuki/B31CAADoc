System Framework:CATSubscriberMethod Typedef

typedef void (CATBaseUnknown::*CATSubscriberMethod)( CATCallbackEvent iPublishedEvent, void * iPublishingObject, CATNotification * iNotification, CATSubscriberData iClientData, CATCallback iCallback)
Prototype of callback methods.
Role: This callback method is declared by means of the method.
Values:
iPublishedEvent
The event in question published by iPublishingObject
iPublishingObject
A pointer to the object that is expected to publish the event
iNotification
The notification published with the event
iClientData
Useful data to pass as argument of iMethodToCall
iCallback
The callback identifier
