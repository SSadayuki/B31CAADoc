System Framework: IClassFactory Interface

Usage: you can freely reimplement this interface.

interface IClassFactory

IClassFactory objects factories are dedicated to instantiating components of a specified CLSID.


Method Index
CreateInstance(IUnknown*,REFIID,void**) Creates an uninitialized object of a specified CLSID.
LockServer(BOOL) Used to keep an out-process server in memory.
