System Framework:CATFChmod Global Function


CATFChmod( iReferenceFile, iFileToUpdate)
Changes the status of a file.
Role: Applies the rights a the file named by iReferenceFile to the file named by iFileToUpdate.
Parameters:
iReferenceFile
the path of the reference file.
iFileToUpdate
the path of the file which rights have to be modified.
Returns:
Legal values:
S_OK :on Success.
negative : on failure
This object is included in the file: CATStdIO.h
