System Framework:CATFileSystemError Typedef

typedef int CATFileSystemError
Definition of the error type for the IO user Exit.
Role: If any error occurs in the code of an IO User Exit, the method must return CATERRNO, in order that the CATIA soft can interpret correctly the error and show the correct error messages.
Values:
0
On success
CATERRNO
if any system error has occurred.
