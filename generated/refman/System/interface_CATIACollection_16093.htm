System Framework: CATIACollection Interface

Usage: you cannot implement this base interface directly. You must first derive it and then implement the derived interface.

interface CATIACollection

Represents the base object for collections.
As a base object, it provides properties shared by any other object. Use the adaptor class to implement this interface.


Method Index
GetItem(CATBSTR&,CATBaseDispatch*&) Retrieves an object with the specified name.
get_Application(CATIAApplication*&) Returns the application.
get_Count(CATLONG&) Returns the number of objects in the collection.
get_Name(CATBSTR&) Returns the name of the collection.
get_Parent(CATBaseDispatch*&) Returns the parent object.
get__NewEnum(IUnknown*&) Returns an iterator on the collection.
