System Framework: CATIUnknownList Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIUnknownList

Interface to handle a list of IUnknown interface pointers.
Role: The first element in the list has index 0.

See also:


Method Index
Add(unsigned int,IUnknown*) Adds an element at a given position in the list.
Count(unsigned int*) Returns the count of elements in the list.
Item(unsigned int,IUnknown**) Returns the iPosition-th element.
Remove(IUnknown*) Removes the first occurence of an element from the list.
Remove(unsigned int) Removes the occurence of an element which position is indicated by iPosition from the list.

Methods


public virtual Add( const iPosition, iItem)

Parameters:

iItem
Element to be added.
Returns:
If iPosition does not respect respect the following rule: 0 <= position < Count, E_FAIL is returned. The element to add must not be a NULL reference or E_POINTER will be returned.
