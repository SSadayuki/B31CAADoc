KinematicsInterfaces Framework: CATIKinCmd Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIKinCmd

Interface to get kinematics command's information.
Role: A kinematics command has a type and drives a kinematics joint. The methods provided by this interface allow to access this information.


Method Index
GetJoint(CATIKinJoint**) Returns the kinematics joint associated to the command.
GetParameter(CATICkeParm**) Returns the parameter of the command.
GetType(char**) Returns the type of the command.
