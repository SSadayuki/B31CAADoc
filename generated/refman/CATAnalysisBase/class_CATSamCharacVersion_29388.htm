public CombineValues()
Combine the values of the collector (if possible) for entities which support repetitions. The CombineValues is possible for a Type of Value : Real (exept for differents Local Axis), Double, Complex and ComplexDouble. AND the RefFrame must be a CATSamRefFrameGlobal.
