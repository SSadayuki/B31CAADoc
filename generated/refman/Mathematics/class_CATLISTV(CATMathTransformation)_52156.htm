Mathematics Framework: CATLISTV(CATMathTransformation) Collection

Usage: you must use this class as is. You should never derive it.

public class CATLISTV(CATMathTransformation)

Collection class for objects representing a mathematical transformation in three dimensions.
Only the following methods of collection classes are available:

  • Append
  • InsertAt
  • RemoveAll
  • RemovePosition
  • Replace
  • ReSize
  • Size
Refer to the articles dealing with collections in the encyclopedia.



This object is included in the file: CATLISTV_CATMathTransformation.h
If needed, your Imakefile.mk should include the module: CATMathematics
