KnowledgeInterfaces Framework: CATIInstanceListener Interface

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIInstanceListener

Interface dedicated to implementing reactions to instances changes.
Role: this interface allows one to be called on instances events, as soon as one has subscribed to such events with .
