CD5SaveOperation_Scope (Enumeration)

enum CD5SaveOperation_Scope {
  CD5SaveOperation_ActiveDocument,
  CD5SaveOperation_CurrentEditor,
  CD5SaveOperation_Session
}
Scope of the Save Operation.
It is used by the CreateSaveOperation function.
Values:
CD5SaveOperation_ActiveDocument
Active document is considered in save scope.
CD5SaveOperation_CurrentEditor
Current editor is considered in save scope.
CD5SaveOperation_Session
All documents in session are considered in save scope.
