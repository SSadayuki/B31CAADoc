PertNode (Object)

This Interface represents a PertNode in the PERT Chart.
Method Index
GetLocation This method gets the Location of the PertNode in the PERT Chart
SetIcon This method sets Icon for the PertNode in the PERT Chart
SetLocation This method place the PertNode in the PERT Chart at any location by specifying the X and Y Coordinates in the Grid.
