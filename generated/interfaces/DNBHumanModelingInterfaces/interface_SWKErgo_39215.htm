SWKErgo (Object)

This interface groups the properties giving access to all the ergonomic studies that can be performed on a manikin.

Its main purpose is to provide bridges to the specific ergonomic analysis interfaces.
Property Index
Carry Returns the object for a carry analysis.
LiftLower Returns the object for a lift/lower analysis.
PushPull Returns the object for a push/pull analysis.
RULA Returns the object for a RULA analysis.

Properties
