SWKLineOfSightNode (Object)

This interface characterizes a node.

A node is a convenient way to regroup different attributes of the manikin.
Method Index

Sub ResetAngularLimitations( piDOFId, piReset)
Reset the angular limitations of the line of sight depending on piReset: 0 -> 2 OR 3 OR 4 depending of the first encountered. 1 -> 2 AND 3 AND 4 2 -> Unlock the value 3 -> Restore the angular limitations if it is "No Limits" 4 -> Set back the angular limitations to their default value (50.0%)
