DrawingArrow (Object)

Represents a drawing arrow in a drawing view.
Property Index
HeadSymbol Returns or sets symbol type of head side.
HeadTarget Returns or sets target element of head side.
NbInterruption Returns the number of interruptions of arrow path.
NbPoint Returns the number of points of arrow path.
ScaleOnExtremities Returns or sets the scale on extremities mode.
TailSymbol Returns or sets symbol type of tail side.
TailTarget Returns or sets target element of tail side.
Method Index
AddInterruption Add an interruption to an arrow.
AddPoint Add a point to an arrow.
GetInterruptions Get arrow path.
GetPoint Get arrow point coordinates.
GetPoints Get arrow path.
ModifyPoint Modify a point of an Arrow.
RemoveInterruption Remove an interruption to an arrow.
RemovePoint Remove a point from an arrow.

Methods


Sub AddInterruption( iFirstPointX, iFirstPointY, iSecondPointX, iSecondPointY)
Add an interruption to an arrow.
Parameters:
iFirstPointX
X coordinates of first point.
iFirstPointY
Y coordinates of first point.
iSecondPointX
X coordinates of second point.
iSecondPointY
Y coordinates of second point.
Example:
This example adds an interruption to MyArrow. 
 iFirstPointX = 10.
 iFirstPointY = 20.
 iSecondPointX = 20.
 iSecondPointY = 20.
 MyArrow.AddInterruption iFirstPointX, iFirstPointY, iSecondPointX, iSecondPointY
Sub AddPoint( iNum, iX, iY)
Add a point to an arrow.
Parameters:
iNum
Point number. Point will be inserted at iNum+1 position.
iX
X coordinates of point to add.
iY
Y coordinates of point to add.
Example:
This example adds a point to MyArrow. 
 iNum = 1
 iX = 10.
 iY = 20.
 MyArrow.AddPoint iNum, iX, iY
