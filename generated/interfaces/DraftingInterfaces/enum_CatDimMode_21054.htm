CatDimMode (Enumeration)

enum CatDimMode {
  catDimClassical,
  catDimCumulate,
  catDimHalfDim,
  catDimChained,
  catDimStacked,
  catDimCumulatesystem,
  catDimHalfDimSystem
}
Dimension mode.
Role: CatDimMode is used to specify dimension mode.
Values:
catDimClassical
Simple dimension.
catDimCumulate,
Cumulate dimension.
catDimHalfDim,
Half dimension.
catDimChained,
Chained dimension.
catDimStacked,
Stacked dimension.
catDimCumulateSystem,
Cumulate dimension system.
catDimHalfDimSystem,
Half dimension system.
