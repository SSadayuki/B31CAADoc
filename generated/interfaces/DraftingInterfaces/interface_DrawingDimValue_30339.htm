DrawingDimValue (Object)

Manages dimension value of a dimension in drawing view.

This interface is obtained from method.

Property Index
FakeDimType Returns or sets fake dimension type of value.
ScoringMode Get dimension scoring mode.
Value Returns value of dimension.
ValueFramedElement Get dimension framed element.
ValueFramedGroup Returns or sets dimension framed group.
