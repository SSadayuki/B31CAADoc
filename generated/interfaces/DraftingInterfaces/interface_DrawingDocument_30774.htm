DrawingDocument (Object)

Represents the Document object for drawings.
Property Index
DrawingRoot Retrieves the drawing root in the drawing document.
Parameters Returns the collection of parameters of the drawing document.
Relations Returns the collection of relations of the drawing document.
Sheets Returns the collection of drawing sheets of the drawing document.
Standard Returns or sets the drawing standard of the drawing document.
Method Index
Isolate Isolates all the drawing views of all the drawing sheets of the drawing document.
Update Updates all the drawing sheets of the drawing document.
