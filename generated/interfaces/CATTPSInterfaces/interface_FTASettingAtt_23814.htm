Sub SetTrueDimensionColorLock( iLocked)
Locks or unlocks the TrueDimensionColor parameter.
Role:Locks or unlocks the TrueDimensionColor parameter if it is possible in the current administrative context. In user mode this method will always return E_FAIL.
Parameters:
iLocked
the locking operation to be performed Legal values:
TRUE : to lock the parameter.
FALSE: to unlock the parameter.
