SchTempListFactory (Object)

Create temporary (non-persistent) lists to be used in VB environment.
Method Index
CreateListOfBSTRs This method returns a new empty list of strings.
CreateListOfDoubles This method returns a new empty list of doubles.
CreateListOfLongs This method returns a new empty list of long integers.
CreateListOfObjects This method returns a new empty list of objects.
