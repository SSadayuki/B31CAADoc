SchGRRFactory (Object)

Factory to create graphical representations of schematic objects.
Method Index
CreateGRRCntr Create the graphical representation of a Schematic Connector.
CreateGRRGroup Create the graphical representation of a Schematic Group.
CreateGRRRoute Create the graphical representation of a Schematic Route.
CreateGRRRouteEllipse Create the graphical representation of a Schematic Route Ellipse.
CreateGRRZone Create the graphical representation of a Schematic Zone.
