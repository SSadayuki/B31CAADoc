SchCntrLocation (Object)

Manage the location of a schematic connector.
Method Index
GetAlignVector Get the current alignment vector of the connector.
GetPosition Get the current position of the connector in absolute coordinates.
GetRelativePosition Get the current position of the connector in relative coordinates.
SetAlignVector Set the current alignment vector of the connector.
SetPosition Set the current position of the connector in absolute coordinates.
SetRelativePosition Set the current position of the connector in absolute coordinates.

Parameters:
