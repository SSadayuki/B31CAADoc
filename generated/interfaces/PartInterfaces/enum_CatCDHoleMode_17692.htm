CatCDHoleMode (Enumeration)

enum CatCDHoleMode {
  catCDModeNoCountersunkDiameter,
  catCDModeCountersunkDiameter
}
Counterdrilled hole mode.
It describes the chosen mode for Counterdrilled hole.
Values:
catCDModeNoCountersunkDiameter
Counterdrilled parameters are Diameter, Depth and Angle
catCDModeCountersunkDiameter
Counterdrilled parameters are Diameter, Depth, Angle and Countersunk Diameter
