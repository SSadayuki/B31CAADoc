HybridShapeSpiral (Object)

Represents the hybrid shape Spiral feature object.
Role: Allows to access data of the Spiral feature. This data includes:
  • type
  • support
  • centre point
  • axis
  • starting radius
  • orientation
  • ending angle
  • ending radius
  • revolution
  • pitch
