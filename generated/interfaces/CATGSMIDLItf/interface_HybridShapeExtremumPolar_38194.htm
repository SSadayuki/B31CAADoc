HybridShapeExtremumPolar (Object)

Represents the hybrid shape extremum polar feature.
Role: To access the data of the extremum polar feature . This data includes:
  • The contour
  • The support (if exist )
  • The direction of evaluation
  • The extermum type

Use the create a HybridShapeExtremumPolar object.

Property Index
Angle returns the resulting angle of extremum.
Contour returns or sets the input contour.
Dir returns or sets the direction of computation.
ExtremumType returns or sets the type of extremum.
Origin returns or sets the origin of the polar axis.
Radius returns the resulting radius of extremum.
Support returns or sets the support (if exist).
