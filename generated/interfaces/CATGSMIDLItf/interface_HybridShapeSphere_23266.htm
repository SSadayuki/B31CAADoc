HybridShapeSphere (Object)

Represents the hybrid shape sphere feature object.
Role: To access the data of the hybrid shape sphere explicit feature object.
The Sphere feature : a Sphere is made up of 4 angles parameters.
See also:
Property Index
Axis Returns or sets axis on the object.
BeginMeridianAngle Returns BeginMeridianAngle on the object.
BeginParallelAngle Returns BeginParallelAngle on the object.
Center Returns or sets the sphere center.
EndMeridianAngle Returns EndMeridianAngle on the object.
EndParallelAngle Returns EndParallelAngle on the object.
Limitation Returns whether the sphere is created as a whole sphere or not.
Radius Role: Get sphere radius.
