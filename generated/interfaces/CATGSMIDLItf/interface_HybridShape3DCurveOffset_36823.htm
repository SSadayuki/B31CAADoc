HybridShape3DCurveOffset (Object)

Represents the hybrid shape 3DCurve Offset operation feature.
Role: Allows you to access data of the 3D Curve Offset feature created by using a curve, a direction and three litteral parameters

Use the to create a HybridShape3DCurveOffset object.

Property Index
CornerRadiusValue Returns or sets the Corner Radius Value.
CornerTensionValue Returns or sets the Corner Tension Value.
CurveToOffset Returns or sets the curve to offset.
Direction Returns or sets the direction.
InvertDirection Returns or sets the direction orientation.
OffsetValue Returns or sets the OffsetValue.
