PspLightConnector (Object)

Represents the light connector.
Role: To access light connector data.
Method Index
GetAlignmentVector Returns the position of the connector.
GetOrientationVector Returns the Orientation Direction of the connector.
GetOrigin Returns the position of the connector.
SetAlignmentVector Sets the alignment direction of the connector.
SetOrientationVector Sets the Orientation Direction of the connector.
SetOrigin Sets the position of the connector.
