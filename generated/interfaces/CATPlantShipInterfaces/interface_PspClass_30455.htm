PspClass (Object)

Represent Interface to list the start up object classes of an application.
Role: Application object classes.
Property Index
StartUpConnectors Returns a List of start-up Connector object classes.
StartUpFunctions Returns a List of start-up Function object classes.
StartUpPhysicals Returns a List of start-up physical object classes.

Properties


Property StartUpConnectors() As (Read Only)
Returns a List of start-up Connector object classes.

  ...
