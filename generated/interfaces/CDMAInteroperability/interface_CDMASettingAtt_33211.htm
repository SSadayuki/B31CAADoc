CDMASettingAtt (Object)

Represents the base object to handle the parameters of the cache.
Property Index
AllowReplaceStructExposedDuringExtractSave Retrieves or sets the enabling of replacing the structure exposed assembly with the blackbox resulting of extract/save.
AskBeforeOverwriteExistingParts Retrieves or sets the enabling of asking before overwriting existing part.
AutoHigh Retrieves or sets the automatic highlight in PSN status.
CheckExistInVPMBeforeFBDI Retrieves or sets the check of existence in VPM before FBDI.
DisableAppObjMgt Retrieves or sets the disabling of applicative object management.
DisableOverwriteReloadWarnMsg Retrieves or sets the desabling of the overwrite/reload message.
DisableVPMSaveCommitPanel Retrieves or sets the disabling of VPM Save Commit Panel.
DisableVPMSaveReportPanel Retrieves or sets the disabling of VPM Save Report Panel.
EnableDirectV5VaultSave Retrieves or sets the enabling of the direct V5 Vault save.
ForbidInconsistentSave Retrieves or sets the enabling of forbidding inconsistent save.
GFBDI Retrieves or sets the enabling of generic FBDI.
LoadAllVPMProperties Retrieves or sets the Load Of All VPM Properties status.
ManualCacheCleaning Retrieves or sets the manual cache cleaning activation.
MappingFile Retrieves or sets the mapping file.
MaxPercentCacheSize Retrieves or sets the maximum percentage cache size before automatic cache cleaning.
NbFilesForDeletion Retrieves or sets the number of files to delete in VPM cache during automatic cache cleaning.
NbFilesMax Retrieves or sets the maximum number of files in VPM cache before automatic cache cleaning.
NeverOverwriteExistingParts Retrieves or sets the enabling of forbidding overwriting existing part.
NoDBConnection Retrieves or sets the database connection desabling.
VPMCachePath Retrieves or sets the VPM cache path.
WorkWithVPMCache Returns or sets the activation of the work with VPM Cache.
