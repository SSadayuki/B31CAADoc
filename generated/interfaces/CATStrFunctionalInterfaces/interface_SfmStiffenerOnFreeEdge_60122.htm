SfmStiffenerOnFreeEdge (Object)

Interface to manage different attributes of Stiffener on free edge.
Role: Provides access to Structural Functional Modeler's Stiffener on free edge.
Property Index
FreeEdgeOffset Returns or sets the SFE's Free Edge offset.
SectionAngle Returns or sets the SFE's section angle.
SideOrientation Returns or sets the side orientation.
Method Index
GetFreeEdge Returns the Free Edge.
GetMoldedSurface Returns the molded surface.
GetWelds Gets Welds feature on operated SFE.
SetFreeEdge Sets the Free Edge.
