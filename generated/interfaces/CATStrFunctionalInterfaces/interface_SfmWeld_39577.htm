SfmWeld (Object)

Interface to manage the Structure Functional Modeler Weld object.
Role: Provides access to each Weld object's attributes.
Property Index
AddedMaterial Gets Added Material.
EdgePreparation Gets Edge Preparation.
FitUp Gets Fit Up.
WeldType Gets Weld Type.

Returns:
