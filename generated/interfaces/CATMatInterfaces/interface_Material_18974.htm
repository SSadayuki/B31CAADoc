Material (Object)

Represents a Material object.
Property Index
AnalysisMaterial Returns the analysis material object from the current material.
RenderingMaterial Returns the rendering material object from the current material.
Method Index
CopyRenderingDataFrom Copy rendering data from a material to the current material.
CreateAnalysisData Create a default analysis material on the current material.
CreateRenderingData Create a default rendering material on the current material.
ExistAnalysisData Returns true if a analysis material exists on the current material.
ExistRenderingData Returns true if a rendering material exists on the current material.
GetIcon Write the icon of a material to disc.
PutIcon Read the icon of a material from JPEG file.
