ManufacturingActivitySyntax (Object)

This interface offers services to manage PP words of activity.
Method Index
ResetPPWORDSyntax Resets the PP words syntax.
SetPPWORDSyntax Sets the PP words syntax.

Methods


Sub ResetPPWORDSyntax()
Resets the PP words syntax.
Returns:
S_OK when the method succeeds, and E_FAIL otherwise
Sub SetPPWORDSyntax( iPPWORDs)
Sets the PP words syntax.
Returns:
S_OK when the method succeeds, and E_FAIL otherwise
Parameters:
iPPWORDs
The user PP words syntax to set
