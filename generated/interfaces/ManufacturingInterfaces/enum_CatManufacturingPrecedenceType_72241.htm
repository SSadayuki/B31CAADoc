CatManufacturingPrecedenceType (Enumeration)

enum CatManufacturingPrecedenceType {
  catPrecedenceTypeJustBefore,
  catPrecedenceTypeBefore
}
Possible types of precedence.
Values:
catPrecedenceTypeJustBefore
The precedence type is JustBefore. No other element can be inserted between the 2 elements.
catPrecedenceTypeBefore
The precedence type is Before. Other elements can be inserted between the 2 elements.
