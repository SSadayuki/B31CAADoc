OriginElements (Object)

Represents the part's 3D reference axis system.
It allows an easy access to 3D reference axis system of a Part object thru the three planes XY, YZ, and ZX.
See for parent object.
Property Index
PlaneXY Returns the XY plane of the part 3D reference axis system.
PlaneYZ Returns the YZ plane of the part 3D reference axis system.
PlaneZX Returns the ZX plane of the part 3D reference axis system.

Properties


Property PlaneXY() As (Read Only)
Returns the XY plane of the part 3D reference axis system.
Example:
The following example returns in plnXY the XY plane of the partRoot part from the partDoc part document: 
 Set partRoot = partDoc.Part
 Set plnXY = partRoot.originElements.PlaneXY
