StrPlate (Object)

Represents a structure plate object.
A plate is defined by a thickness and aggregates a support object and a sketch defining its contour.
Property Index
Offset Returns the offset between the given support and the support of the plate.
Sketch Returns the sketch object of the plate.
StandardThickness Returns the standard thickness following the support orientation.
Support Returns or sets the support of the plate.
Method Index
ReverseDirection Reverses the material orientation of the plate.
