CatVisLayerType (Enumeration)

enum CatVisLayerType {
  catVisLayerBasic,
  catVisLayerNone
}
Types of layer.
Role: This enum is used in the interface.
Values:
catVisLayerBasic
basic layer
This type of layer gets a number from 1 to 1000.
catVisLayerNone
"None" layer
An object which has a "None" layer does not have any specific layer.
